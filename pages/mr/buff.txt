====== Buff Base Class - Code References ======

Machine-readable reference page for the Buff base class entity.

===== Entity Information =====
  * **Entity Kind:** Buff
  * **Type:** Base class for all buff/debuff effects
  * **Implementation:** Java abstract class

===== Java Classes =====
  * [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/actors/buffs/Buff.java|Buff.java]] - Abstract base class for all buffs
  * [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/actors/buffs/BuffIndicator.java|BuffIndicator.java]] - Buff visual indicator constants
  * [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/actors/buffs/BuffFactory.java|BuffFactory.java]] - Factory for creating buff instances
  * [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/actors/hero/Hero.java#L1200|Hero.java]] - Buff management in hero class
