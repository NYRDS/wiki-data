==== Code Implementation ====

```java
public class Albino extends Rat {
    {
        hp(ht(15));
    }

    @Override
    public int attackProc(@NotNull Char enemy, int damage) {
        if (Random.Int(2) == 0) {
            Buff.affect(enemy, Bleeding.class).level(damage);
        }
        return damage;
    }
}
```
