====== Rogue Class - Machine Reference ======

This page provides machine-readable reference data for the Rogue hero class in Remixed Dungeon.

===== Overview =====
The Rogue is one of the base hero classes in Remixed Dungeon, specializing in stealth, evasion, and surprise attacks. The Rogue starts with a dagger and has access to the Smoke Bomb special ability through their class armor. When choosing a mastery path, Rogues can become either an [[mr:assassin_subclass|Assassin]] or a [[mr:freerunner_subclass|Freerunner]].
