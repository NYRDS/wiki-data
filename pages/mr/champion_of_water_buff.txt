====== Champion of Water Buff - Code References ======

**Champion of Water** is a buff implemented in Lua that enhances a creature's resilience and fortitude, making it a stalwart adversary capable of enduring relentless attacks.

===== Java Registration =====

**Buff Factory Registration**:
  * **Class**: `com.nyrds.pixeldungeon.mechanics.buffs.BuffFactory`
  * **Constant**: `CHAMPION_OF_WATER = "ChampionOfWater"`
  * **Source**: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/nyrds/pixeldungeon/mechanics/buffs/BuffFactory.java#L72|BuffFactory.java#L72]]

This buff is registered in the BuffFactory and instantiated via the Lua scripting system. No dedicated Java class exists for this buff.

===== JSON Configuration =====
No specific JSON configuration file found for this buff. Buff configuration is handled through the Lua script and string resources.
