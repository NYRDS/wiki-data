====== Frozen Fish Item - Code References ======

**Frozen Fish** is a food item in Remixed Dungeon. This page contains machine-readable reference information for the frozen_fish_item entity.

===== Entity Kind =====
  * **Entity Kind**: frozen_fish
  * **Entity Type**: item
  * **Wiki Page**: [[en:rpd:frozen_fish_item|Frozen Fish Item]]

===== Lua Implementation =====
  * **Script File**: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/assets/scripts/items/FrozenFish.lua|FrozenFish.lua]]
  * **Script Location**: RemixedDungeon/src/main/assets/scripts/items/FrozenFish.lua
  * **Implementation Details**:
    * Image: 14 (from items/food.png sprite sheet)
    * Stackable: true
    * Price: 15 gold
    * Name String Key: FrozenFish_Name
    * Info String Key: FrozenFish_Info
    * **onThrow**: Spawns Piranha mob if thrown in water, otherwise drops as item
    * **burn**: Transforms into FriedFish when burned
