====== Pixel Dungeon ML Project ======

Pixel Dungeon ML (известный также как Pixel Dungeon RU) проект задачей которого является перевод Pixel Dungeon на максимальное количество языков с минимально возможными изменениями в коде оригинала.

===== Цель проекта =====

Целью проекта Pixel Dungeon ML является перевод игры Pixel Dungeon на максимально возможное количество языков при минимальных изменениях в оригинальном коде.

===== Описание =====

Этот проект служит для локализации игры Pixel Dungeon. Он позволяет адаптировать игру под различные языки и культуры, обеспечивая при этом сохранение игрового баланса и функциональности.
