====== Левитация ======

{{ rpd:images:levitation_buff.png|Левитация }}

**Левитация** - положительный статусный эффект в Remixed Dungeon, который позволяет персонажу парить над землей.

==== Описание эффекта ====
  * **Тип:** Положительный статусный эффект (Бафф)
  * **Эффект:** Позволяет парить над уровнем земли
  * **Продолжительность:** 20 ходов (Levitation.DURATION = 20f)
  * **Источник:** В основном из [[ru:rpd:potion_of_levitation_item|Зелья левитации]]

==== Получение ====
  * Из выпитого [[ru:rpd:potion_of_levitation_item|Зелья левитации]]
  * Из определённых заклинаний или классовых способностей
  * Из специальных предметов или эффектов окружения

==== Механика ====
  * Позволяет перемещаться по ловушкам без их активации
  * Даёт возможность проходить по специальным поверхностям, которые обычно затрудняют передвижение
  * Предотвращает воздействие некоторых эффектов, связанных с землёй
  * Блокирует эффект [[ru:rpd:roots_buff|Корни]] (отсоединяет существующие корни)
  * Когда эффект заканчивается, вызывается Dungeon.level.press(), что может активировать эффекты пола на позиции персонажа
  * Спрайт персонажа показывается как CharSprite.State.LEVITATING
  * Использует BuffIndicator.LEVITATION для иконки в интерфейсе

==== Стратегия ====
  * Отлично подходит для безопасного прохождения областей с ловушками
  * Позволяет обходить определённые препятствия и сложные участки местности
  * Полезно для избегания вредных эффектов, связанных с землёй
  * Хорошо для навигации по опасным клеткам пола
  * Важно рассчитать время окончания эффекта, чтобы не приземлиться в опасной зоне
