====== Гладиатор ======

{{ rpd:images:warrior_gladiator_hero.png|Гладиатор }}

==== Описание ====
**Гладиатор** — это один из двух [[ru:rpd_mastery|путей Мастерства]] для класса [[ru:rpd_warrior_class|Воин]], доступный после победы над [[ru:rpd_tengu_mob|Тенгу]] с помощью [[ru:rpd_tome_of_mastery_item|Фолианта Мастерства]].

Гладиаторы специализируются на комбо-атаках с оружием ближнего боя. Каждый успешный удар накапливает комбо, которое усиливает следующие атаки.

==== Особенности ====
  * **Комбо-атаки:** При использовании ближнего оружия накапливается комбо, увеличивающее урон последующих ударов
  * **Классовая броня:** [[ru:rpd_gladiator_armor_item|Доспех Гладиатора]] — дает способность «Рывок»
  * **Стиль игры:** Агрессивный ближний бой с накоплением урона

==== Способность класса — Рывок ====
Доспех Гладиатора дает способность «Рывок» (Heroic Leap):
  * Герой прыгает к выбранной клетке
  * При приземлении оглушает всех врагов в соседних клетках
  * Доступно только Гладиаторам

==== Получение ====
  1. Победите [[ru:rpd_tengu_mob|Тенгу]] (босс 2-го уровня)
  2. Получите [[ru:rpd_tome_of_mastery_item|Фолиант Мастерства]]
  3. Прочитайте Фолиант и выберите «Гладиатор»
