====== Книга Заклинаний ======

{{ rpd:images:spell_book_item.png|Книга Заклинаний }}

**Книга Заклинаний** — это артефакт в Remixed Dungeon, который позволяет героям изучать и использовать различные магические заклинания. Она служит хранилищем для различных магических способностей, которые можно использовать в бою, для утилит или поддержки.

==== Описание ====
Книга Заклинаний выглядит как стопка случайных страниц, сшитых вместе, а не как настоящая книга. Но при должном внимании можно вычленить из неё полезные заклинания.

==== Особенности ====
  * **Хранение заклинаний:** Может хранить несколько заклинаний для использования во время игры
  * **Доступность:** Позволяет героям получать доступ к заклинаниям, даже если они не имеют природной склонности к ним
  * **Обучение:** Предоставляет способ изучать новые заклинания помимо классовых склонностей
  * **Мана:** На использование заклинаний тратит [[ru:rpd_mana|Ману]]
  * **Содержимое:** В книге одно из двух заклинаний

==== Заклинания ====
Книга может содержать различные заклинания:

  * **Волшебный факел** — заклинание, которое производит свет, озаряющий тёмные уровни подземелья
  * **Исцеление** — незаменимое заклинание для каждого путешественника. Восстанавливает 1/3 от максимального запаса здоровья заклинателя

==== Категории заклинаний ====
  * **[[ru:rpd_combat_mechanic|Боевые заклинания]]:** Заклинания для прямого урона и боевой утилиты
  * **[[ru:rpd_utility_mechanic|Утилитарные заклинания]]:** Заклинания для перемещения, побега или взаимодействия с окружением
  * **[[ru:rpd_support_mechanic|Поддерживающие заклинания]]:** Заклинания лечения и усиления

==== Классы и склонности ====
  * [[ru:rpd_spell_affinities|Склонности к заклинаниям]] определяют, какие заклинания наиболее эффективны для каждого класса
  * [[ru:rpd_hero_class|Классы героев]] могут иметь природные склонности к определённым типам заклинаний
  * [[ru:rpd_hero_subclass|Подклассы]] могут открывать специализированный доступ к заклинаниям

==== Стратегия ====
  * Учитывайте класс героя при выборе заклинаний для изучения
  * Некоторые заклинания синергируют со специфическими способностями класса
  * Определённые ситуации могут требовать конкретных типов заклинаний (например, урон против утилиты)
