====== Рубиновый арбалет ======

{{ rpd:images:ruby_crossbow_item.png|Рубиновый арбалет}}

**Рубиновый арбалет** - это мощное [[ru:rpd:ranged_weapon|оружие дальнего боя]] в Remixed Dungeon. Это смертоносное оружие действительно является чудом дворфийской инженерии.

==== Обзор предмета ====
  * **Тип:** Оружие дальнего боя (Арбалет) - [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/nyrds/pixeldungeon/items/guts/weapon/ranged/RubyCrossbow.java|RubyCrossbow.java]]
  * **Урон:** Высокий диапазон урона (зависит от реализации базового класса Crossbow)
  * **Прочность:** Прочный (как другие арбалеты)
  * **Вес:** Умеренный вес для оружия такого типа
  * **Цена:** Значительная цена (информация уточняется)
  * **Требуется СИЛ:** 20 (как указано в описании)
  * **Скорость:** Медленная скорость атаки по сравнению с другими типами оружия

==== Особые механики ====
  * **Дальнобойная атака:** Позволяет атаковать врагов на расстоянии без вступления в [[ru:rpd:melee_weapon_item|ближний бой]]
  * **Высокий урон:** Наносит высокий урон при каждой атаке
  * **Малоэффективен в ближнем бою:** Для всех классов, кроме [[ru:rpd:elf_class|эльфов]], эффективность в ближнем бою снижена
  * **Эльфийская специфика:** [[ru:rpd:elf_class|Эльфы]] могут наносить нормальный урон этим арбалетом даже в ближнем бою без стрел

==== Стратегия ====
  * Идеален для дальнего боя и уничтожения одиночных сильных врагов
  * Особенно эффективен для [[ru:rpd:elf_class|эльфов]] благодаря способности наносить нормальный урон в ближнем бою
  * Лучше всего использовать на открытых пространствах, где можно использовать преимущества дальнего боя
  * Не рекомендуется использовать в тесных помещениях против большого количества врагов, если не играешь за [[ru:rpd:elf_class|эльфа]]

==== Получение ====
  * Может быть найден на различных уровнях подземелья
  * Может встречаться в [[ru:rpd:ammo|сундуках]] и других контейнерах
  * Информация о точных источниках уточняется

==== Сравнение с другими арбалетами ====
  * **Преимущества:**
    * Высокий урон
    * Специальные свойства эльфийского ближнего боя
    * Высокое качество изготовления (дворфийская инженерия)
  * **Недостатки:**
    * Медленная скорость атаки
    * Низкая эффективность в ближнем бою (для всех классов, кроме [[ru:rpd:elf_class|эльфов]])
