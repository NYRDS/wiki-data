====== Корона Крысиного Короля ======

{{ rpd:images:rat_king_crown_item.png|Корона Крысиного Короля }}

==== Описание ====
Золотая Корона Его Крыссности, символ его власти. Это [[en:rpd_artifacts|артефакт]], который позволяет вам подчинять крыс в бою.

==== Свойства ====
  * **Тип**: [[en:rpd_artifacts|Артефакт]]
  * **Эффект**: При ударе по [[ru:rpd_rat_mob|крысе]] она не получает урон, а становится вашим [[en:rpd_allies|союзником]]
  * **Особенности**: Не работает на Псевдо-Крыс

==== Механика ====
  * При попадании по [[ru:rpd_rat_mob|крысе]] во время боя, крыса становится вашим союзником
  * Эффект действует только на обычных крыс, не на боссов или специальных версий
  * Предоставляет постоянный [[ru:rpd_buffs|бафф]] "Крыссность"

==== Стратегия ====
  * Полезно для создания армии из крыс в определенных ситуациях
  * Лучше использовать на уровнях с высокой плотностью крыс
  * Не забывайте, что эффект работает только при прямом попадании по крысе
