====== Рубиновый лук ======

{{ rpd:images:ruby_bow_item.png|Рубиновый лук }}

**Рубиновый лук** — уникальное [[ru:rpd:weapon|оружие]] в Remixed Dungeon, мощный лук для экспертов.

==== Общая информация ====
  * **Тип:** [[ru:rpd:ranged_weapon|Оружие]] дальнего боя
  * **Классификация:** [[ru:rpd:bow_item|Лук]]
  * **Редкость:** Очень редкий
  * **Слот:** Основная рука (оружие дальнего боя)
  * **Дальность:** 5 клеток
  * **Урон:** 5 (базовый)
  * **Скорость:** 0.8 (медленный)
  * **Точность:** 1.5 (очень высокая)

==== Особые свойства ====
  * Почти каждая выпущенная из него стрела пронзает врага насквозь, оставляя огромные дыры в его плоти
  * Требует значительной силы для эффективного использования
  * Очень сложно натянуть тетиву

==== Стратегия ====
  * Превосходен для опытных игроков с высокими характеристиками
  * Идеален для атаки нескольких врагов подряд благодаря пронзающим выстрелам
  * Требует соответствующего уровня силы для использования без штрафов
  * Превосходное оружие для дистанционной бойни на средней дистанции

==== Получение ====
  * Может быть найден как редкая добыча с сильных врагов
  * Иногда встречается в сундуках сокровищ
  * Может быть результатом [[ru:rpd:alchemy_mechanic|алхимических]] рецептов

==== Сравнение ====
  * Сильно превосходит по урону обычные луки
  * Уступает по скорости другим видам оружия
  * Идеален для стрелковых персонажей с высокими характеристиками
