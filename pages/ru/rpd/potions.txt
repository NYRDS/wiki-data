====== Зелья ======

==== Описание ====
**Зелья** - это одноразовые расходные предметы, которые при использовании оказывают различный эффект на героя. Зелья можно использовать из инвентаря или бросить на землю для создания эффекта в точке падения.

==== Типы зелий ====
  * [[ru:rpd:potion_of_healing_item|Зелье исцеления]] - Восстанавливает здоровье
  * [[ru:rpd:potion_of_experience_item|Зелье опыта]] - Даёт уровень опыта
  * [[ru:rpd:potion_of_strength_item|Зелье силы]] - Повышает силу героя
  * [[ru:rpd:potion_of_might_item|Зелье могущества]] - Повышает силу и здоровье
  * [[ru:rpd:potion_of_mind_vision_item|Зелье зоркости]] - Показывает расположение врагов
  * [[ru:rpd:potion_of_purity_item|Зелье чистоты]] - Устраняет негативные эффекты
  * [[ru:rpd:potion_of_invisibility_item|Зелье невидимости]] - Делает героя невидимым
  * [[ru:rpd:potion_of_liquid_flame_item|Зелье жидкого пламени]] - Создаёт огонь при броске
  * [[ru:rpd:potion_of_paralytic_gas_item|Зелье парализующего газа]] - Создаёт парализующий газ при броске
  * [[ru:rpd:potion_of_toxic_gas_item|Зелье токсичного газа]] - Создаёт токсичный газ при броске
  * [[ru:rpd:potion_of_frost_item|Зелье холода]] - Создаёт эффект заморозки при броске

==== Использование ====
  * Зелья можно использовать напрямую из инвентаря
  * Зелья можно бросать на врагов или на пол для создания эффекта в точке падения
  * Некоторые зелья имеют разные эффекты при прямом использовании и при броске

==== Механика ====
  * Зелья могут иметь разные эффекты в зависимости от взаимодействия с окружающей средой
  * [[ru:rpd:alchemy|Алхимия]] позволяет создавать зелья из семян и других ингредиентов

==== Стратегия ====
  * Сохраняйте ценные зелья на критические ситуации
  * Используйте зелья в бою с боссами для получения преимущества
  * Бросайте агрессивные зелья (огонь, газ) на группы врагов
