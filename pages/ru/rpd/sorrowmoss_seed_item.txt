====== Семена Грустномоха ======

{{ rpd:images:sorrowmoss_seed_item.png|Семена Грустномоха }}

**Семена Грустномоха** — это предмет, получаемый из растения Грустномох. Их можно использовать для посадки новых растений или в алхимии для создания зелий.

==== Описание ====
Семена собираются с растения Грустномох. Их можно посадить на плодородную почву для выращивания новых растений Грустномох или использовать в алхимическом котле для создания Зелья Токсичного Газа.

==== Свойства ====
  * **Тип:** Семена
  * **Результат алхимии:** При использовании 3 и более семян в алхимическом котле может создаться Зелье Токсичного Газа
  * **Посадка:** Можно сажать на плодородную почву для выращивания Грустномоха
  * **Вес:** Очень лёгкие

==== Использование ====
  * **Посадка:** Использовать на плодородной почве для выращивания Грустномоха
  * **Алхимия:** Соединить 3 и более семян в алхимическом котле для попытки создания Зелья Токсичного Газа
  * **Замачивание:** Можно замочить зельями воды для создания особых предметов

==== Алхимия ====
  * 3+ семян Грустномоха в алхимическом котле могут дать Зелье Токсичного Газа
  * Большее количество семян увеличивает шанс создания конкретного зелья (Токсичного Газа)

==== Стратегия ====
  * Храните для алхимии, чтобы создавать полезные зелья Токсичного Газа
  * Будьте осторожны при сборе с растений Грустномох
  * Полезны для создания зелий с поражающим эффектом
