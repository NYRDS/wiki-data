====== Свиток снятия проклятия ======

{{ rpd:images:item_scroll_of_remove_curse.png|Свиток снятия проклятия }}

**Свиток снятия проклятия** — магический [[en:rpd:scroll|свиток]] в Remixed Dungeon, который удаляет эффекты проклятий с предметов.

===== Общая информация =====
  * **Тип:** [[en:rpd:scroll|Свиток]]
  * **Редкость:** Нечастый
  * **Специальный эффект:** Удаляет проклятия с [[en:rpd:equipment_mechanic|снаряжения]]
  * **Использование:** Одноразовый предмет
  * **Индекс спрайта:** ItemSpriteSheet.SCROLL_YNGVI (43)

===== Особые механики =====
  * Удаляет статус проклятия с [[en:rpd:weapon|оружия]], [[en:rpd:armor|брони]], [[en:rpd:ring|колец]] и другого [[en:rpd:equipment_mechanic|снаряжения]]
  * Позволяет снять проклятое [[en:rpd:equipment_mechanic|снаряжение]] нормально после экипировки
  * Рассеивает [[en:rpd:invisibility_buff|Невидимость]] с героя при использовании
  * Удаляет дебафф [[en:rpd:weakness_buff|Слабости]] с героя
  * Влияет на все проклятые предметы в инвентаре героя одновременно

===== Стратегия =====
  * Необходим для борьбы с проклятым снаряжением, которое нельзя снять
  * Полезен перед [[en:rpd:scroll_of_upgrade_item|улучшением]] снаряжения для обеспечения наилучших результатов
  * Важно носить при изучении более глубоких уровней подземелья, где проклятые предметы более распространены
  * Можно использовать в качестве профилактики на подозрительных предметах

===== Получение =====
Свитки снятия проклятий можно найти:
  * На обычных [[en:rpd:level|уровнях подземелья]]
  * В специальных сокровищницах
  * Потенциально для покупки в магазинах
  * В качестве наград за выполнение заданий
  * Относительно часто встречаются в магазинах по сравнению с другими специализированными свитками

===== Использование =====
  * Прочитайте свиток, чтобы активировать его эффект
  * Все проклятые предметы в инвентаре героя будут очищены от проклятия
  * [[en:rpd:invisibility_buff|Невидимость]] героя будет снята (если активна)
  * Дебафф [[en:rpd:weakness_buff|Слабости]] героя будет удален (если активен)
