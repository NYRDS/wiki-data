====== Посохи ======

{{ rpd:images:wand_of_magic_missile.png|Посох}}

**Посохи** - это магическое [[ru:rpd:equipment|снаряжение]] в Remixed Dungeon, которое может быть использовано как оружие ближнего боя, но в основном применяется для [[ru:rpd:spell|заклинаний]] с расходом зарядов.

==== Типы посохов ====
  * [[ru:rpd:wand_of_magic_missile_item|Посох магических стрел]] - начальный посох [[ru:rpd:mage_class|Мага]]
  * [[ru:rpd:wand_of_lightning_item|Посох молнии]]
  * [[ru:rpd:wand_of_disintegration_item|Посох дезинтеграции]]
  * [[ru:rpd:wand_of_firebolt_item|Посох огня]]
  * [[ru:rpd:wand_of_ice_item|Посох холода]]
  * [[ru:rpd:wand_of_regrowth_item|Посох роста]]
  * [[ru:rpd:wand_of_smoke_bomb_item|Посох дымовой бомбы]]
  * [[ru:rpd:wand_of_flock_item|Посох стада]]
  * [[ru:rpd:wand_of_teleportation_item|Посох телепортации]]
  * [[ru:rpd:wand_of_poison_item|Посох яда]]
  * [[ru:rpd:wand_of_amok_item|Посох буйства]]
  * [[ru:rpd:wand_of_slowness_item|Посох замедления]]
  * [[ru:rpd:wand_of_shielding_item|Посох щита]]
  * [[ru:rpd:wand_of_warding_item|Посох охраны]]
  * [[ru:rpd:wand_of_blink_item|Посох моргания]]
  * [[ru:rpd:wand_of_summoning_item|Посох призыва]]

==== Механика посохов ====
  * **Заряды:** Посохи имеют ограниченное количество зарядов, которые восстанавливаются с [[ru:rpd:resting|отдыхом]]
  * **Улучшение:** Можно улучшить с помощью [[ru:rpd:scroll_of_upgrade_item|Свитка улучшения]]
  * **Идентификация:** Неизвестный посох требует [[ru:rpd:identification|идентификации]] для определения типа
  * **Классовая синергия:** [[ru:rpd:mage_class|Маг]] получает бонусы при использовании посохов

==== Использование ====
  * Нажмите на посох в панели действий чтобы использовать
  * Наведите на цель чтобы прицелиться (если применимо)
  * Посохи можно использовать как оружие ближнего боя

==== Эффективность ====
  * Уровень посоха увеличивает его эффективность
  * [[ru:rpd:mage_class|Маг]] получает бонусы от [[ru:rpd:spell_power|магической силы]] для всех посохов
  * Некоторые посохи имеют уникальные свойства

==== Получение ====
  * Посохи можно найти в сундуках, у врагов или [[ru:rpd:shops|магазинах]]
  * [[ru:rpd:mage_class|Маг]] начинает с уникальным [[ru:rpd:wand_of_magic_missile_item|Посохом магических стрел]]
