====== Кристалл хаоса ======

{{ rpd:images:chaos_crystal_item.png|Кристалл хаоса }}

**Кристалл хаоса** - это артефакт в Remixed Pixel Dungeon, который можно надеть как кольцо. Он обладает способностью «Метка хаоса» для создания случайных эффектов.

==== Описание ====
Кристалл хаоса - уникальный артефакт из набора хаоса, который сочетает в себе свойства кольца и инструмента для создания случайных эффектов.

==== Способность: Метка хаоса ====
Чтобы использовать способность «Метка хаоса», нужно накопить заряды в кристалле:
  * Кристалл набирает заряды, когда герой получает урон
  * Метка хаоса может создать случайный эффект на выбранной клетке
  * Возможные эффекты включают: поджог зоны, выпуск облака газа и другие случайные эффекты

==== Заряды ====
  * **Максимум зарядов**: 100
  * **Минимум для слияния**: 50 зарядов
  * Для фиксации количества зарядов кристалл должен быть опознан

==== Слияние ====
Если опознать кристалл повторно (при наличии минимум 50 зарядов), можно слить его с другими предметами:
  * Со [[ru:rpd:scrolls|свитком]] → получаем [[ru:rpd:scroll_of_enchantment_weapon_item|свиток зачарования оружия]]
  * С [[ru:rpd:armor|бронёй]]
  * С [[ru:rpd:weapons|оружием ближнего или дальнего боя]]
  * С [[ru:rpd:wands|жезлом]]

==== Получение ====
Кристалл хаоса можно получить:
  * Из сундуков в ледяных пещерах
  * Как награду за выполнение квестов хаоса
  * Через алхимическое создание

==== Стратегия ====
  * Копите заряды перед использованием способности
  * Используйте в бою для создания тактических преимуществ
  * Слияние с ценными предметами может создать мощные комбинации
