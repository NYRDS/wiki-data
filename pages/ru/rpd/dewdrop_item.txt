====== Росинка ======

{{ rpd:images:dewdrop_item.png|Росинка }}

==== Описание ====
**Росинка** (Dewdrop) - это мелкий потребляемый предмет в Remixed Dungeon, который предоставляет небольшое количество исцеления при употреблении. Это кристально чистая росинка, которая дает небольшое исцеление при употреблении.

==== Механика ====
  * **Тип:** Потребляемый (Еда)
  * **Исцеление:** Восстанавливает 5-10 HP
  * **Вес:** 0 (без веса)
  * **Стакается:** Да
  * **Расчет исцеления:** `value = 1 + (Dungeon.depth - 1) / 5` (глубина подземелья влияет на исцеление)

==== Приобретение ====
  * Найдена при раздавливании высокой травы
  * Шанс увеличивается с Кольцом Травничества
  * Иногда можно найти в сундуках
  * Может выпадать с некоторых врагов

==== Использование ====
  * Может быть употреблена напрямую для исцеления
  * Предоставляет экстренное лечение в опасных ситуациях
  * Лучше, чем ничего, когда здоровье низкое

==== Стратегия ====
Росинки полезны для небольшого исцеления и должны собираться при нахождении. Хотя они не обеспечивают значительного исцеления, они могут спасти жизнь в экстренных случаях, когда нет других лечебных предметов.
