====== Печень Тенгу ======

{{ rpd:images:tengu_liver_item.png|Печень Тенгу }}

**Печень Тенгу** - это специфическая награда в Remixed Dungeon, которую получает [[ru:rpd:gnoll_class|Гнолл]] после победы над [[ru:rpd:tengu_mob|Тенгу]].

==== Функция ====
  * **Награда для Гнолла:** Выдается только когда Гнолл побеждает Тенгу
  * **Специальный трофей:** Уникальная награда для класса Гнолл
  * **Редкий предмет:** Можно получить только один раз за игру

==== Получение ====
  * **[[ru:rpd:tengu_mob|Тенгу]]:** Выбрасывает печень когда убит [[ru:rpd:gnoll_class|Гноллом]]

==== Стратегия ====
  * Особая награда для игроков, выбравших класс [[ru:rpd:gnoll_class|Гнолл]]
  * Собирательный трофей для фанатов этого класса
  * Не имеет функционального использования, но имеет коллекционную ценность
