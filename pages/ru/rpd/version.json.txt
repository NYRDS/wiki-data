====== Version.json Configuration File ======

Файл **version.json** - это конфигурационный файл, используемый для определения метаданных модов в Remixed Dungeon.

==== Структура файла ====

<code javascript>
{
  "version":6,
  "name": "Cool Mod Name",
  "author": "Proud Mod Developer",
  "description":"Detailed mod description",
  "url": "https://wiki.nyrds.net/",
  "hr_version":"version 6!",
  "rpd_version":610
}
</code>

==== Описание полей ====

  * **version** - Машиночитаемая версия мода, совершенно обязательно
  * **name** - Имя мода, как оно показывается игроку, обязательно
  * **rpd_version** - версия Remixed Dungeon нужная чтобы запускать это мод, обязательно
  * **author** - Разработчики мода, как будет показано игроку, опционально
  * **url** - Кликабельный линк на сайт разработчиков, опционально
  * **hr_version** - человеко-читаемая версия мода, опционально

==== Значение ====

Этот файл важен при создании модификаций для Remixed Dungeon, так как позволяет системе правильно идентифицировать и управлять модами.
