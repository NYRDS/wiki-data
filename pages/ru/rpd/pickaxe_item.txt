====== Кирка ======

{{ rpd:images:pickaxe_item.png|Кирка}}

==== Описание ====
**Кирка** - это квестовый инструмент, который можно получить от Кузнеца-Тролля в Пещерах. Она используется для добычи [[en:rpd:dark_gold_item|Мракозлата]].

==== Характеристики ====
  * **Тип**: Инструмент/Оружие
  * **Редкость**: Квестовый предмет
  * **Вес**: 3
  * **Урон**: 1-8 (схожий с боевым посохом)

==== Механика ====
  * **Добыча**: Используется для добычи [[en:rpd:dark_gold_item|Мракозлата]] из руды в Пещерах
  * **Не требует экипировки**: Не обязательно экипировать в слот оружия для добычи
  * **Кровавая кирка**: При убийстве летучей мыши начинает мерцать красным цветом
  * **Квест**: Исчезает при сдаче квеста Кузнецу

==== Получение ====
  * Получается от Кузнеца-Тролля в Пещерах как часть квеста
  * Является обязательным элементом для выполнения квеста Кузнеца

==== Использование ====
  * Добывает [[en:rpd:dark_gold_item|Мракозлато]] из рудных жил в Пещерах
  * Может использоваться как оружие (если экипирована)
