====== Механика запрещенных действий ======

Механики игры позволяют ограничивать определенные действия для некоторых классов героев. Некоторые классы не могут выполнять определенные действия в игре.

==== Список возможных действий для запрета ====
  * ''Scroll_ACRead'' - Чтение свитков
  * ''Bag_ACOpen'' - Открытие сумок
  * ''DewVial_ACDRINK'' - Питье из сосуда с росой
  * ''Drink_ACDrink'' - Выпивание зелья
  * ''EquipableItem_ACEquip'' - Экипировка предметов
  * ''EquipableItem_ACUnequip'' - Снятие экипировки
  * ''Item_ACDrop'' - Выбрасывание предметов
  * ''Item_ACThrow'' - Бросание предметов
  * ''TomeOfMastery_ACRead'' - Чтение фолианта мастерства
  * ''Torch_ACLight'' - Зажигание факелов
  * ''Weightstone_ACApply'' - Применение гирь
  * ''Amulet_ACEnd'' - Завершение игры с Амулетом Йендера
  * ''ArmorKit_ACAplly'' - Применение набора брони
  * ''Pickaxe_ACMine'' - Добыча шахтерской киркой
  * ''Plant_ACPlant'' - Посадка семян
  * ''Potion_ACMoisten'' - Увлажнение предметов зельями
  * ''ShortSword_ACReforge'' - Перековка коротким мечом
  * ''Wand_ACZap'' - Использование жезлов
  * ''WandOfMagicMissile_ACDisenchant'' - Расчарование жезлом магических стрел

==== Использование ====
Эти действия могут быть ограничены в конфигурационном файле ''InitHeroes.json'' для конкретных классов героев.

==== Проверка содержимого ====
  * **Источник информации**: Конфигурация в файле ''RemixedDungeon/src/main/assets/hero/initHeroes.json''
  * **Проверка механики**: Проверено на соответствие реальной реализации в игре
  * **Последнее обновление**: Февраль 2026
