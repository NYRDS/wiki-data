====== Боевой доспех ======

{{ rpd:images:warrior_armor_item.png|Боевой доспех }}

**Боевой доспех** - это знаменательная броня класса Воина в Remixed Dungeon. Несмотря на тяжелый внешний вид, он позволяет Воину совершать богатырский рывок в направленное место, с размаху оглушая всех соседних врагов.

==== Описание ====
Боевой доспех - это уникальная броня, доступная только классу Воина. Она сочетает в себе защиту тяжелой брони с возможностью совершения героического прыжка.

==== Свойства ====
  * **Тип брони**: Тяжелая броня
  * **Класс**: Только для Воина
  * **Особое действие**: Богатырский рывок
  * **Класс предмета**: ClassArmor (специфична для класса Воина)

==== Механики ====
Боевой доспех имеет особое умение под названием **Богатырский рывок**:
  * Позволяет Воину прыгнуть в направленное место
  * При приземлении оглушает всех соседних врагов
  * Это делает его очень эффективным для распыления групп врагов и создания пространства для Воина
  * Время оглушения: 3 хода
  * Время прыжка: 1 ход

==== Эксклюзивность класса ====
  * Только класс Воина может носить боевой доспех
  * Попытки других классов экипировать его будут неудачными с сообщением: "Этот доспех доступен лишь воину!"

==== Варианты ====
У боевого доспеха есть несколько продвинутых вариантов:
  * [[en:rpd:gladiator_armor_item|Гладиаторский доспех]] - Продвинутый воинский доспех
  * [[en:rpd:berserker_armor_item|Доспех берсерка]] - Продвинутый воинский доспех

==== Стратегия ====
  * Используйте богатырский рывок для инициации боя с группами врагов
  * Оглушение дает вам преимущество в первом ударе
  * Комбинируйте с другими умениями Воина для максимального эффекта
  * Помните, что прыжок тратит ход, поэтому планируйте заранее
