====== Броня Гнолла ======

{{ rpd:images:gnoll_armor_item.png|Броня Гнолла }}

==== Описание ====
Броня Гнолла (официально называемая "мундир гнолл-лейтенанта") - это классовая броня для класса героя Гнолл. Это одна из специальных классовых бронь, которую можно получить, используя [[ru:rpd:armor_kit_item|Набор Брони]] на любой обычной броне. Броня Гнолла имеет уникальную специальную способность "ПОДЧИНИТЬ", которая позволяет владельцу очаровывать близлежащих врагов и превращать их в питомцев, но только если владелец является героем-гноллом.

==== Свойства ====
  * **Класс Брони:** Уникальная классовая броня для героя-гнолла
  * **Специальная Способность:** ПОДЧИНИТЬ - очаровывает близлежащих врагов, превращая их в питомцев
  * **Ограничение Класса:** Только герой-гнолл может экипировать эту броню
  * **Не Улучшается:** Не может быть зачарована или улучшена
  * **Прочность:** Стандартная прочность брони
  * **Изображение:** Индекс 25 в спрайт-листе armor.png

==== Специальная Способность: ПОДЧИНИТЬ ====
При активации способность ПОДЧИНИТЬ пытается очаровать всех видимых врагов в поле зрения и в пределах досягаемости (до ShadowCaster.MAX_DISTANCE). Количество мобов, которые могут быть подчинены, ограничено уровнем владельца (максимум - уровень владельца / 6). Очарованные враги становятся верными питомцами и будут сражаться за игрока.

==== Лор ====
Согласно описанию в игре: "Обладатели Гнолл-Лейтенантских мундиров хорошо известны своим умением убеждать самых диких монстров присягнуть Короне". Эта броня представляет командное присутствие гнолл-лейтенантов в их обществе.

==== Получение ====
Броню Гнолла можно получить, используя [[ru:rpd:armor_kit_item|Набор Брони]] на любой обычной броне, но она доступна только игрокам, которые разблокировали класс героя Гнолл.
