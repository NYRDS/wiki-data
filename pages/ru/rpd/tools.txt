====== Инструменты разработки Remixed Dungeon ======

==== Описание ====
Вот инструменты, которые используются при работе над Remixed Dungeon:

==== Используемые инструменты ====
  * **Android Studio** - IDE для разработки Android-приложений
  * **GIMP** - Графический редактор для работы с изображениями и спрайтами
  * **tiled** - Редактор тайловых карт, используется для создания уровней
