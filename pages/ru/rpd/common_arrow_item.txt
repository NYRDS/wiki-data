====== Обычная стрела ======

{{ rpd:images:common_arrow_item.png|Обычная стрела }}

**Обычная стрела** — это базовый тип стрел в Remixed Dungeon.

==== Описание ====
Обычные стрелы являются наиболее распространённым типом боеприпасов для луков и арбалетов. Они не имеют дополнительных эффектов, но наносят базовый урон.

==== Характеристики ====
  * **Тип:** Боеприпасы
  * **Урон:** Зависит от используемого лука или арбалета
  * **Редкость:** Обычная
  * **Использование:** Одноразовая (расходуется при выстреле)

==== Использование ====
  * Можно найти в подземелье
  * Можно [[ru:rpd:moisten|смазать]] зельями для получения магических эффектов
  * Наносит урон только если выпущена из лука или арбалета
  * Базовый боеприпас для всех типов луков

==== Получение ====
Обычные стрелы можно получить:
  * Найти в сундуках на различных уровнях подземелья
  * Купить у торговца
  * Получить в качестве награды за квесты
  * Найти в трупах врагов

==== Тактика ====
  * Эффективны против большинства врагов
  * Рекомендуется носить с собой запас стрел
  * Можно смазать зельями для получения дополнительных эффектов
  * Хороший выбор для ранней игры

  * [[ru:rpd:frost_arrow_item|Ледяная стрела]] - Стрела с эффектом заморозки
  * [[ru:rpd:poison_arrow_item|Отравленная стрела]] - Стрела с эффектом яда
