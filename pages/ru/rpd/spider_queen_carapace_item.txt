====== Панцирь Королевы Пауков ======

{{ rpd:images:spider_queen_carapace_item.png|Панцирь Королевы Пауков }}

==== Описание ====
Панцирь Королевы Пауков — редкий материал для крафта, получаемый от [[ru:rpd:spider_queen_mob|Королевы Пауков]]. Это осколок панциря Королевы Пауков с кусочком паучьей железы внутри, который кажется живым.

==== Свойства ====
  * **Тип предмета**: Материал для крафта
  * **Складываемый**: Да
  * **Редкость**: Очень редкий (дроп с босса)
  * **Источник**: [[ru:rpd:spider_queen_mob|Королева Пауков]] (выпадает после победы)

==== Применение ====
  * В настоящее время рецепты крафта неизвестны (будущий контент)
  * Потенциальный материал для экипировки с паучьей тематикой
  * Может использоваться для создания брони со свойствами паутины

==== Примечания ====
Этот материал высоко ценится за свои уникальные свойства, содержит живую паучью железу, которая, кажется, всё ещё активна. Его ищут игроки, которые хотят создать продвинутую экипировку с паучьей тематикой.
