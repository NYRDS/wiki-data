====== Тип передвижения ======

**Тип передвижения** (Walking Type) - это свойство мобов, определяющее, по каким поверхностям они могут передвигаться в Remixed Dungeon.

==== Типы передвижения ====
  * **NORMAL** - обычное передвижение, только по проходимым клеткам
  * **WATER** - передвижение по воде
  * **WALL** - передвижение сквозь стены (например, [[ru:rpd:shadow_mob|Тени]])
  * **GRASS** - передвижение по траве
  * **CHASM** - передвижение над пропастями
  * **ABSOLUTE** - абсолютное передвижение, может перемещаться по любой клетке

==== Применение ====
  * Тип передвижения определяет, где моб может появиться при спауне
  * Влияет на алгоритм поиска пути при передвижении моба
  * Определяет возможность передвижения по определённым типам клеток
