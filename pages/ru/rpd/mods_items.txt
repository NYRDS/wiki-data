
====== Предметы ======

===== Аттрибуты предметов =====

==== Общие ====
  * "quantity" - количество ( натуральное число - 1 или больше ), по умолчанию 1
  * "level" - уровень качества предмета (целое число), по умолчанию 0
  * "cursed" - проклят ли предмет (true|false), по умолчанию нет
  * "identified" - опознан ли предмет (true|false), по умолчанию нет

==== Аттрибуты оружия ====
  * "enchantment" - зачарование наложенное на оружие ( Fire, Poison, Death, Paralysis, Leech, Slow, Swing, Piercing, Instability, Horror, Luck ), по умолчанию отсутствует

==== Аттрибуты брони ====
  * "glyph" - глиф наложенный на броню (Bounce, Affection, AntiEntropy, Multiplicity, Potential, Metabolism, Stench, Viscosity, Displacement, Entanglement)

==== Аттрибуты палочек ====
  * "charges" - текущее количество зарядов в палочке
  * "maxCharges" - максимальное количество зарядов в палочке
