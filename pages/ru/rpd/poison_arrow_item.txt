====== Ядовитая Стрела (Poison Arrow) ======

{{ rpd:images:poison_arrow_item.png|Poison Arrow }}

**Ядовитая Стрела** — отравленное метательное оружие, которое накладывает эффект яда на врага при попадании.

==== Описание ====
Наконечник этой стрелы блестит каплей смертельного яда.

==== Свойства ====
  * **Урон**: 3-4
  * **Задержка**: 0.75
  * **Цена**: 5 золота за единицу
  * **Вес**: 1
  * **Тип**: Метательное оружие

==== Эффект ====
  * При попадании накладывает эффект [[ru:rpd:poison_buff|Отрава]] на 5 секунд по умолчанию
  * Длительность яда увеличивается при использовании с усиленным оружием
  * Отравленная цель получает урон с течением времени

==== Где найти ====
  * Сундуки сокровищ в различных уровнях подземелья
  * Можно создать путём смачивания [[ru:rpd:common_arrow_item|обычной стрелы]] с помощью [[ru:rpd:potion_of_toxic_gas_item|Зелья Токсичного Газа]]
