====== Паутина ======

==== Описание ====
Паутина - это ловушка, встречающаяся по всему подземелью, которая может обездвижить персонажей. Когда персонаж наступает на местность с паутиной, он застревает и не может двигаться до тех пор, пока не освободится или пока паутина не будет удалена.

==== Свойства ====
  * **Тип**: Ловушка/Местность
  * **Эффект**: Обездвиживает персонажа
  * **Длительность**: До разрушения или удаления
  * **Взаимодействие**: Может быть очищена с помощью определенных предметов или способностей

==== Механики ====
  * Персонажи, наступающие на местность с паутиной, обездвиживаются
  * Некоторые доспехи, такие как [[ru:rpd:spider_armor_item|Паучья броня]], позволяют свободно передвигаться по местности с паутиной
  * Может быть очищена с помощью огненных атак или другими методами
  * Распространяется при определенных условиях

==== Стратегия ====
  * Будьте осторожны при перемещении по зонам с паутиной
  * Рассмотрите возможность взять с собой огненные предметы для очистки паутины
  * Паучья броня позволяет безопасно перемещаться по зонам с паутиной
