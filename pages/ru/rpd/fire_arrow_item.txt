====== Предмет: Огненная стрела ======

{{ rpd:images:fire_arrow_item.png|Огненная стрела }}

==== Описание ====
**Огненная стрела** - это специальный вид [[ru:rpd:arrow_item|стрел]] в Remixed Dungeon, который накладывает эффект [[ru:rpd:burning_buff|Горения]] на цель при попадании. Эти стрелы особенно эффективны против врагов, чувствительных к огню.

==== Статистика ====
  * **Тип:** Амуниция (расходуемый предмет)
  * **Урон:** Стандартный для стрел
  * **Эффект:** Накладывает [[ru:rpd:burning_buff|Горение]] на цель
  * **Наносит урон:** Да
  * **Взаимодействие с огнём:** Нельзя использовать с [[ru:rpd:bow_item|луками]], имеющими зачарование [[ru:rpd:fire_enchantment|Огонь]]
  * **Цена:** Зависит от количества (обычно 5-10 золотых за единицу)
  * **Вес:** 1/4 (небольшой вес)

==== Получение ====
  * **Алхимия** - можно создать, используя [[ru:rpd:moisten|смачивание]] обычных [[ru:rpd:arrow_item|стрел]] с [[ru:rpd:potion_of_liquid_flame_item|Зельем жидкого пламени]]
  * **Сокровищницы** - могут быть найдены в сундуках или на этажах
  * **Крафт** - через алхимические рецепты

==== Использование ====
  * **С [[ru:rpd:bow_item|луками]]** - эффективно использовать с луками для дальнего боя
  * **[[ru:rpd:mechanics|Механика]]** - при попадании накладывает [[ru:rpd:burning_buff|Горение]]
  * **[[ru:rpd:combat|Бой]]** - особенно эффективны против групп врагов
  * **[[ru:rpd:tactical|Тактика]]** - использовать для поджога врагов перед ближним боем

==== Стратегия ====
  * **Наиболее эффективна** против врагов без сопротивления огню
  * **Можно использовать** для поджога области перед входом в бой
  * **Совместимость** - несовместима с огненными зачарованиями на оружии
  * **Стоит учитывать** - эффект [[ru:rpd:burning_buff|Горения]] может повредить вам, если вы слишком близко
