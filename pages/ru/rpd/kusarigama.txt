====== Кусаригама ======

{{ rpd:images:kusarigama_item.png|Кусаригама }}

**Кусаригама** - это уникальное оружие ближнего боя 3 уровня, сочетающее серп с утяжелённой цепью. Это оружие имеет специальную способность притягивания и визуальные эффекты цепи при атаке с расстояния.

==== Характеристики ====
  * **Тип оружия:** [[ru:rpd:melee_weapons|Оружие ближнего боя]]
  * **Уровень:** 3
  * **Урон:** 2-8 (в среднем 5, масштабируется по уровню)
  * **Прочность:** 100%
  * **Вес:** 1,0 (нормальный)
  * **Скорость:** 1,0x (нормальная скорость)
  * **Слоты [[ru:rpd:enchantments|Зачарований]]:** 1

==== Особые свойства ====
  * **Визуализация цепи:** При атаке с расстояния отображает визуальный эффект цепи, соединяющей пользователя с целью
  * **Способность притягивания:** Может притягивать врагов или объекты к владельцу с помощью цепи
  * **Эффект головокружения:** 10% шанс наложить [[ru:rpd:vertigo_buff|Головокружение]] при попадании (запутывает цель)
  * **Анимация:** Использует класс анимации меча

==== Механика улучшения ====
  * Можно [[ru:rpd:upgrading|улучшить]] с помощью [[ru:rpd:scroll_of_upgrade_item|Свитков улучшения]]
  * Можно применить [[ru:rpd:enchantments|зачарования]] для добавления специальных эффектов
  * Улучшения увеличивают урон и прочность
  * Максимальный уровень улучшения: +10

==== Получение ====
Кусаригаму можно найти:
  * [[ru:rpd:dungeon|В подземелье]] - Случайные выпады оружия
  * [[ru:rpd:shops|В лавках]] - Доступна для покупки
  * [[ru:rpd:chests|В сундуках]] - Как случайная добыча

==== Стратегия ====
  * Эффективно для героев, предпочитающих ближний бой с утилитой
  * Способность притягивания может использоваться для перемещения врагов
  * Визуальный эффект цепи обеспечивает обратную связь при атаке с расстояния
  * Эффект головокружения может запутать врагов, заставляя их атаковать случайно
  * Рассмотрите зачарования, которые дополняют эффекты замешательства

==== Сравнение ====
По сравнению с другим [[ru:rpd:weapons|оружием]]:
  * Стандартный урон оружия 3 уровня
  * Уникальная утилита притягивания, не встречающаяся на другом оружии
  * Визуальный эффект цепи отличает его от стандартного оружия ближнего боя
  * 10% шанс головокружения добавляет возможность контроля толпы

==== Проверка содержания ====
  * Java-класс: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/items/weapon/melee/Kusarigama.java|Kusarigama.java]]
  * Kind сущности: Kusarigama (возвращает простое имя класса через getEntityKind())
  * Проверка характеристик: Извлечено напрямую из конструктора Kusarigama.java `super(3, 2f, 1f)` - уровень 3, множитель урона 2,0, множитель скорости 1,0
  * Особая механика: На основе анализа кода Kusarigama.java

    - Эффект головокружения: 10% шанс в методе `postAttack()`

  * Регистрация фабрики предметов: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/nyrds/pixeldungeon/items/common/ItemFactory.java|ItemFactory.java]] включает Kusarigama в генерацию оружия
