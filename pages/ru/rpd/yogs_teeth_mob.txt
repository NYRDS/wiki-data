====== Пасть Йога ======

{{ rpd:images:yogs_teeth_mob.png|Пасть Йога }}

==== Описание ====
**Пасть Йога** (Yog's Teeth) — мощный моб, встречающийся на уровне Глубины Глу, представляющий часть битвы с Йог-Джевой. Это один из второстепенных боссов, которые могут появиться в битве с Йогом.

==== Базовые Статы ====
  * **Здоровье**: 350 (hp(ht(350)))
  * **Макс. Здоровье**: 350
  * **Базовый Навык Атаки**: 46
  * **Базовый Навык Защиты**: 44
  * **Урон**: 50-80 (dmgMin = 50, dmgMax = 80)
  * **Сопротивление Урону**: 21 (dr = 21)
  * **Опыт за Убийство**: 26 (expForKill = 26)
  * **Может быть Питомцем**: Нет (canBePet() возвращает false)

==== Специальные Способности ====
  * **Похищение Жизни**: При 1/3 шансе исцеляется на нанёсённый урон (Random.Int(3) == 1: heal(damage, this))
  * **Кровотечение**: При 1/3 шансе накладывает кровотечение при попадании (Random.Int(3) == 1: Buff.affect(enemy, Bleeding.class).level(damage))
  * **Двойной Урон**: При 1/3 шансе наносит двойной урон (Random.Int(3) == 1: Devour.hit(enemy); return damage*2)
  * **Тревожный Призыв**: При получении урона оповещает всех мобов на уровне о своей позиции (damage вызывает mob.beckon(getPos()) для всех мобов)
  * **Иммунитеты**: Иммунен к [[ru:rpd_paralysis_buff|Параличу]], [[ru:rpd_amok_buff|Амоку]], [[ru:rpd_sleep_buff|Сну]], [[ru:rpd_terror_buff|Ужасу]] и [[ru:rpd_burning_buff|Горению]] (вызовы addImmunity())
  * **Сопротивления**: Сопротивляется [[ru:rpd_toxic_gas_buff|Токсичному Газу]] (вызов addResistance())

==== Информация о Появлении ====
  * **Уровень**: Глубины Глу (уровень 22-24)
  * **Тип**: Второстепенный босс (один из нескольких возможных в битве с Йогом)
  * **Связанные Мобы**: Часть битвы с Йог-Джевой вместе с [[ru:rpd_yogs_brain_mob|Мозг Йога]], [[ru:rpd_eye_mob|Око]], [[ru:rpd_yogs_heart_mob|Сердце Йога]]
  * **Редкость**: Редкое появление в битве с Йог-Джевой

==== Боевая Стратегия ====
  * Высокий урон требует осторожного позиционирования
  * Способность похищения жизни делает продолжительный бой сложным
  * Прока двойного урона может быть разрушительной, поэтому могут потребоваться защитные меры
  * Победа над этим мобом — часть выполнения испытания Йог-Джевы
  * Старайтесь победить его быстро, прежде чем он значительно исцелится за счёт похищения жизни
  * Рассмотрите использование эффектов контроля толпы несмотря на иммунитет к некоторым (Паралич, Сон и т.д.)
