====== Бумеранг ======

{{ rpd:images:boomerang_sprite.png|Бумеранг }}

**Бумеранг** - изогнутое метательное оружие, которое возвращается к метателю после броска. Это уникальное оружие дальнего боя с возвращающимися свойствами.

==== Описание ====
При броске врагу это плоское, изогнутое, деревянное снаряд возвратится в руки метателя. Охотница начинает с этим уникальным улучшаемым оружием.

==== Характеристики ====
  * **Урон:** 1-4 (базовый) - масштабируется с улучшениями
  * **Требуемая сила:** 10
  * **Прочность:** Возвращается после использования (фактически неограниченная)
  * **Специальное свойство:** Возвращается после броска
  * **Цена:** 100 золота
  * **Можно улучшать:** Да
  * **Эффект износа:** MIN -= 1, MAX -= 2 за уровень износа
  * **Время экипировки:** 0 (мгновенная экипировка)
  * **Тип полета:** Криволинейный путь, быстрое вращение

==== Особые свойства ====
  * **Возвращение:** Автоматически возвращается к метателю после попадания в цель или промаха
  * **Дальнее сражение:** Можно использовать для атаки дальних врагов
  * **Многоразовый:** Не требует извлечения как другие метательные оружия
  * **Можно улучшать:** Можно улучшать как и другое оружие
  * **Не для перековки:** Нельзя использовать с действием перековки короткого меча
  * **Траектория полета:** Летит по изогнутому пути, а не по прямой (isFliesStraight() возвращает false)
  * **Вращение:** Быстро вращается в полете (isFliesFastRotating() возвращает true)

==== Улучшение ====
  * Каждое улучшение: MIN += 1, MAX += 2
  * Единственное метательное оружие, которое можно улучшать

==== Зачарования ====
  * Не может естественным образом получить зачарования Пронзания или Размаха (метод зачарования фильтрует эти)
  * Другие зачарования работают нормально

==== Заметки ====
  * Бумеранг - стартовое оружие для класса [[rpd:huntress|Охотницы]]
  * Он автоматически возвращается в инвентарь игрока или быстрые слоты после броска
  * Имеет специальные механики для возврата в слот инвентаря/быстрого слота после использования
  * Это оружие уникально тем, что сочетает возможности дальнего боя с неограниченной прочностью
  * Единственное метательное оружие, которое начинает игру уже экипированным
  * Автоматически экипируется в слот инвентаря, откуда было брошено
  * Если он был в быстром слоте, он возвращается в этот быстрый слот после броска
