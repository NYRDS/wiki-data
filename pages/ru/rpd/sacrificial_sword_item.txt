====== Меч для жертвоприношений ======

{{ rpd:images:sacrificial_sword_item.png|Меч для жертвоприношений }}

Был введён в версии [[changelog|remix.21.3.alpha.5]]

===== Внутриигровое описание =====

«На первый взгляд, этот старый бронзовый меч с потёками патины совершенно бесполезен. Но внутри него спят какие-то древние, архаичные силы, способные превратить почти любое существо в безвольного раба. Нужно лишь найти способ их пробудить...»

===== Характеристики =====

Меч для жертвоприношений - оружие ближнего боя 2 порядка.

  * **Базовый урон:** 2
  * **Скорость атаки:** 1.0x (нормальная)
  * **Точность:** 1.0x (нормальная)
  * **Требуемая сила:** 13 (базовое, масштабируется с улучшениями)
  * **Прочность:** Обычная прочность оружия
  * **Зачарование:** Не может быть зачарован
  * **Можно улучшить:** Да

===== Особые свойства =====

При атаке врага Мечом для жертвоприношений есть шанс обратить врага в питомца, который сражается за игрока:

  * **Формула шанса обращения:**
    * Базовый шанс = (0.25 + (уровень героя × 4 + уровень меча²) × 0.01) × бонус класса / штраф брони
    * Штраф брони = DR брони цели / 8
    * Бонусы класса: Некромант (×3.0), Шаман (×2.0), Страж (×1.5), остальные (×1.0)
    * Дополнительные модификаторы: - (навык защиты врага × 0.01 × скорость врага) - (HP врага × 0.01)

===== Ограничения =====

  * Не может обращать боссов
  * Не может обращать NPC
  * Работает только на мобах
  * Меньший шанс против врагов с высоким навыком защиты и скоростью
  * Меньший шанс против врагов с высоким HP
  * Срабатывает только при прямых попаданиях (не при промахах)

===== Механика =====

  * Меч испускает красное свечение (0xFF4466), которое пульсирует с периодом max(0.1, 0.1/базовыйШанс), что увеличивается с ростом шанса обращения
  * Свечение пульсирует быстрее с увеличением шанса обращения
  * Успешное обращение врага запускает механику создания [[ru:rpd:pet_mechanic|питомца]]
  * Не работает на врагах, которые уже являются питомцами

===== Стратегия =====

  * Носите лёгкую броню или не носите её вообще для максимизации шанса обращения (DR брони уменьшает шанс успеха)
  * Подклассы героя, такие как Шаман или Некромант, имеют значительно более высокие шансы обращения
  * Постарайтесь ослабить врагов перед попыткой обращения (меньше HP увеличивает шансы)
  * Избегайте использования против быстрых врагов с высокой защитой
  * Пульсирующее свечение может визуально указывать текущий шанс обращения
  * Рассмотрите использование дебаффов для уменьшения защиты врага перед попыткой обращения
