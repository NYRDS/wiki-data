====== Сообщение Заточенного Кобольда 2 ======

{{ rpd:images:caged_kobold_mob.png|Заточенный Кобольд }}

==== Текст ====
Приятель, ты должен помочь мне

==== Контекст ====
Это одно из сообщений, которое НПС [[en:rpd_caged_kobold_mob|Заточенный Кобольд]] может сказать при взаимодействии.

==== Описание ====
Заточенный Кобольд просит о помощи у героя этим сообщением. Это часть последовательности диалогов при взаимодействии с захваченным кобольдом в [[ru:rpd_ice_caves|Ледяных пещерах]].

==== Местоположение ====
Находится в [[ru:rpd_ice_caves|Ледяных пещерах]] при встрече с [[ru:rpd_nip_npc|Заточенным Кобольдом]].
