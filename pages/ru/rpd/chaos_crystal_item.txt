====== Кристалл Хаоса (Предмет) ======

{{ rpd:images:chaos_crystal_item.png|Кристалл Хаоса }}

**Кристалл Хаоса** — это артефакт в Remixed Pixel Dungeon, обладающий способностью «Метка хаоса». Для использования способности необходимо сначала зарядить кристалл (он заряжается при получении урона героем). Метка хаоса может создавать случайные эффекты в выбранной области, такие как поджигание территории или выпуск облака газа и т.п. Если идентифицирован, можно зафиксировать количество зарядов (до 100 максимум). Если полностью идентифицирован, можно слить его с предметом (требуется минимум 50 зарядов).

==== Описание ====
Кристалл Хаоса — странный и мощный артефакт, канализирующий хаотические энергии. Его можно носить как кольцо, и он имеет две основные способности: Метку хаоса и слияние предметов. Кристалл заряжается, когда герой получает урон, что делает его предметом с высоким риском и высокой наградой.

==== Характеристики ====
  * **Тип артефакта:** Кольцо-подобный артефакт
  * **Действия:** Использование, Слияние
  * **Зарядка:** Кристалл заряжается при получении урона героем
  * **Максимум зарядов:** 100 (при идентификации)
  * **Требование для слияния:** Минимум 50 зарядов и двойная идентификация
  * **Предметы для слияния:**
    * Свитки
    * Броня
    * Оружие ближнего боя
    * Жезлы
    * Щиты

==== Механика ====
  * Заряды увеличиваются при получении урона героем (каждый случай урона добавляет 1 заряд, до 100)
  * **Метка хаоса:** При использовании с зарядами создаёт случайный хаотический эффект в выбранной области
  * **Слияние:** Можно сливать с различными предметами для создания тематических версий хаоса:
    * Свиток + Кристалл → Свиток улучшения оружия
    * Оружие + Кристалл → Оружие хаоса (Меч, Лук и т.д.)
    * Броня + Кристалл → Броня хаоса
    * Жезл + Кристалл → Посох хаоса
    * Щит + Кристалл → Щит хаоса
  * Если проклят, использование Метки хаоса нацелится на позицию героя вместо выбранной области
  * Кристалл имеет случайный светящийся цвет при экипировке

==== Использование ====
Кристалл Хаоса можно использовать для:
  * Создания случайных эффектов в стратегических местах с помощью способности Метка хаоса
  * Слияния с предметами для создания мощного тематического оборудования хаоса
  * Любителей риска, которые могут накапливать заряды, стратегически получая урон
  * Универсального предмета, который может улучшать другое оборудование через слияние

==== Стратегия ====
Как эффективно использовать Кристалл Хаоса:
  * Заряжайте кристалл, стратегически получая урон (например, используя броню с передачей урона)
  * Используйте способность метки хаоса для создания благоприятных эффектов в бою
  * Сливайтесь с предметами высокого уровня для лучших тематических замен хаоса
  * Будьте осторожны при проклятии — метка хаоса нацелится на вас вместо желаемого места
  * Случайный светящийся эффект делает его визуально отличительным в инвентаре
