====== Броня Знахаря ======

{{ rpd:images:witchdoctor_armor_item.png|Броня Знахаря }}

Броня Знахаря - это уникальная броня подкласса, получаемая путём использования [[ru:rpd:armor_kit_item|Набора брони]] на любой обычной броне для подкласса [[ru:rpd:witchdoctor_subclass|Знахарь]] класса [[ru:rpd:gnoll_class|Гнолл]] в Remixed Dungeon.

==== Получение ====
  * Используйте [[ru:rpd:armor_kit_item|Набор брони]] на любой обычной броне, играя за [[ru:rpd:gnoll_class|Гнолла]], выбравшего подкласс [[ru:rpd:witchdoctor_subclass|Знахарь]]
  * Преобразует базовую броню в Броню Знахаря с особыми свойствами

==== Особые свойства ====
  * Наследует все свойства [[ru:rpd:gnoll_armor_item|Брони гнолла]]
  * Эксклюзивна для подкласса [[ru:rpd:witchdoctor_subclass|Знахарь]] - нельзя экипировать, если вы не подкласс Знахаря

==== Использование ====
  * Необходимая броня для максимальной эффективности подкласса [[ru:rpd:witchdoctor_subclass|Знахарь]]
  * Должна быть создана как можно скорее после выбора подкласса [[ru:rpd:witchdoctor_subclass|Знахарь]]
  * Позволяет полностью использовать способности подкласса

==== Примечания ====
  * Часть системы идентификации подклассов в Remixed Dungeon
  * Броня подклассов представляет уникальную природу каждого подкласса
  * Не может быть надета персонажами, которые не разблокировали соответствующий подкласс
  * Наследует свойства и особую способность базовой [[ru:rpd:gnoll_armor_item|Брони гнолла]]
