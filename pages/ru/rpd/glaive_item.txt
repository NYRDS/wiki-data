====== Глефа ======

{{ rpd:images:glaive_item.png|Глефа }}

**Глефа** — древковое оружие 5 порядка в Remixed Dungeon. Позволяет атаковать врагов с расстояния благодаря удлинённому радиусу атаки.

==== Описание ====
Глефа — древковое оружие с длинным древком и острым лезвием на конце. Сочетает преимущества копья (дальность атаки) с режущей силой клинка, что делает её эффективным оружием для атаки врагов на расстоянии.

==== Свойства Оружия ====
  * **Тип урона:** Режущий урон с преимуществом дальности
  * **Скорость:** Медленная скорость атаки
  * **Точность:** Хорошая точность
  * **Дальность:** Удлинённая дальность позволяет атаковать врагов с большего расстояния
  * **Улучшение:** Можно улучшить с помощью [[en:rpd:scroll_of_upgrade_item|Свитков Улучшения]]
  * **Зачарования:** Совместима с [[ru:rpd:enchantments|зачарованиями оружия]]

==== Характеристики ====
  * **Порядок:** 5
  * **Урон:** 5-28 (базовый урон до улучшений)
  * **Скорость:** 1.4x скорость атаки (медленнее среднего)
  * **Точность:** 1.2x точность (выше среднего)
  * **Дальность:** Удлинённая дальность (радиус 2)
  * **Требование Силы:** 18 (базовое требование, уменьшается с улучшениями)

==== Механика ====
  * **Удлинённая Дальность:** Может атаковать врагов с расстояния
  * **Свойства Древкового Оружия:** Специальная механика, связанная с дальностью
  * **Преимущество Дальности:** Атакуйте врагов, оставаясь вне зоны контратаки
  * **Двуручное:** Требует две руки для эффективного использования

==== Стратегия ====
  * Отлично подходит для контроля коридоров и узких мест
  * Позволяет сражаться с врагами, сохраняя безопасную дистанцию
  * Эффективно в узких пространствах, где враги не могут окружить вас
  * Медленная скорость требует тщательного позиционирования, чтобы избежать ответной атаки
  * Лучше всего использовать на открытых участках, где можно максимизировать преимущество дальности

==== Получение ====
  * Глефы можно найти по всему подземелью
  * Может быть доступна в магазинах, в зависимости от уровня
  * Относительно редка из-за специализированного характера
