====== Чистый свиток ======

{{ rpd:images:blank_scroll_item.png|Чистый свиток }}

Чистый свиток - это просто свиток, чистый, без руны. Такие иногда можно найти в подземелье, особенно часто они встречаются в библиотеках.

Предмет этот однако не бесполезный, с помощью [[ru:rpd:arcane_stylus_item|Мистического Стилуса]] на нём можно начертать руну, стилус сам выберет какую.

==== Свойства ====
  * **Редкость:** Нечастый
  * **Вес:** 5
  * **Идентификация:** Всегда идентифицирован
  * **Ограничения по классу:** Может использоваться всеми классами героев

==== Функция ====
  * **Преобразование Мистическим Стилусом:** При использовании с [[ru:rpd:arcane_stylus_item|Мистическим Стилусом]], стилус начертит случайную руну на чистом свитке, преобразуя его в случайный свиток
  * **Трансмутация:** Может быть преобразован в случайный свиток броском в Источник Трансмутации
  * **Преобразование Зельем Невидимости:** Зелье Невидимости может быть использовано для 'смазывания' до 3 свитков в одном слоте инвентаря в чистые свитки

==== Специальные свойства ====
  * [[ru:rpd:suspicious_rat_mob|Подозрительная крыса]] - может появиться при использовании чистого свитка рядом с ящиком, сундуком или тайником
  * Мистический Стилус не расходуется при начертании руны на чистом свитке
  * Может быть использован для создания свитка улучшения начертанием руны улучшения
  * Несколько чистых свитков с одинаковой руной могут быть объединены в один стек

==== Стратегия ====
  * Сохраняйте чистые свитки для использования с Мистическим Стилусом, когда вам нужен конкретный тип свитка
  * Полезно для преобразования в полезные свитки, когда вам нужны определённые эффекты
  * Рассмотрите использование с источниками трансмутации, если вам нужны конкретные типы свитков

==== Получение ====
  * Находится как случайная добыча на уровнях подземелья
  * Чаще встречается в библиотеках
  * Создаётся использованием Зелья Невидимости на обычных свитках
