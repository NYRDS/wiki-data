====== Оружие ближнего боя ======

{{ rpd:images:sword_item.png|Оружие ближнего боя }}

**Оружие ближнего боя** в Remixed Dungeon предназначено для ближнего боя. Используется в непосредственном бою с врагами на короткой дистанции.

==== Описание ====
Оружие ближнего боя - это предметы экипировки, которые герой держит в основной руке для боя на короткой дистанции. Они различаются по скорости атаки, урону и особым свойствам.

==== Свойства ====
  * **Навык атаки:** Бонус к шансу попадания в бою
  * **Диапазон урона:** Минимальный и максимальный урон, наносимый при ударе
  * **Прочность:** Сколько использований до разрушения оружия (если применимо)
  * **Фактор скорости:** Насколько быстро можно использовать оружие в бою
  * **Зачарование:** Специальный магический эффект (для улучшенного оружия)

==== Типы оружия ближнего боя ====
  * [[en:rpd:sword_item|Мечи]] - Многофункциональное клинковое оружие
  * [[en:rpd:dagger_item|Кинжалы]] - Быстрое, легкое колющее оружие
  * [[en:rpd:axe_item|Топоры]] - Оружие с тяжелым ударом
  * [[en:rpd:mace_item|Булавы]] - Дробящее оружие с сильным уроном
  * [[en:rpd:spear_item|Копья]] - Пронзающее оружие с досягаемостью
  * [[en:rpd:war_hammer|Боевой молот]] - Тяжелое дробящее оружие
  * [[en:rpd:quarterstaff|Алебарда]] - Сбалансированное двуручное оружие
  * [[en:rpd:kusarigama_item|Кусаригама]] - Коса с цепью

==== Боевая стратегия ====
  * **Высокий урон против Скорости:** Выберите между оружием с тяжелым ударом или быстрыми атаками
  * **Универсальность:** Некоторое оружие лучше работает с определенными [[en:rpd:hero_class|классами героев]]
  * **[[en:rpd:enchantments|Зачарования]]:** Подумайте, какие магические свойства принесут пользу вашему стилю боя
  * **Прочность:** Некоторое оружие более устойчиво к износу

==== Особые свойства ====
  * **Блок:** Некоторое оружие может блокировать атаки (например, [[en:rpd:war_hammer|Боевой молот]])
  * **Досягаемость:** Некоторое оружие может атаковать соседних врагов (например, [[en:rpd:spear_item|Копье]])
  * **Сила:** Для эффективного использования некоторых видов оружия требуется минимальная сила

==== Улучшения ====
  * Оружие ближнего боя можно [[en:rpd:upgrade|улучшать]] с помощью [[en:rpd:scroll_of_upgrade_item|Свитков улучшения]]
  * [[en:rpd:enchantments|Зачарования]] можно применять для получения особых эффектов
  * Более высокие уровни обеспечивают лучшие характеристики
