====== Уровень: Канализация ======

{{ rpd:images:pedestal_level_object.png|Уровень Канализация }}

**Канализация** (Sewers) - это первый уровень в Remixed Dungeon, состоящий из 5 этажей (глубин 1-5). Здесь игрок знакомится с базовой механикой игры.

==== Особенности уровня ====
  * **Тема:** Грязные подземелья с водой, крысами и гноллами
  * **Враги:** Крысы, гноллы, крабы, мухи, слизи
  * **Босс:** **[[ru:rpd:goo_mob|Goo]]** (на 5-м этаже)
  * **Ловушки:** Стандартные ловушки (яды, дротики, тревога)
  * **Секреты:** Скрытые двери, тайники за фальшивыми стенами

==== Ключевые механики ====
  * **Вода:** Можно пить для восстановления сытости, но может быть отравлена
  * **Трава:** Можно сжечь для создания огня или использования в алхимии
  * **Ключи:** Железные ключи открывают запертые двери и сундуки

==== Награда за босса ====
При победе над Goo игрок получает:
  * **[[ru:rpd:goo_drop|Награда Goo]]** - случайный уникальный предмет
  * Доступ к следующему уровню - **[[ru:rpd:prison_level|Тюрьма]]**

==== Стратегия ====
  * Исследуйте каждый этаж полностью для опыта и лута
  * Берегите зелья исцеления для боя с боссом
  * Используйте окружение (воду, траву) в своих целях
  * Собирайте ключи для доступа к запертым зонам
