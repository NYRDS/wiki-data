====== Алхимия ======

{{ rpd:images:potion_of_healing_item.png|Зелье исцеления }}

**Алхимия** (Alchemy) - это система крафта в Remixed Dungeon, позволяющая создавать зелья из ингредиентов (семян, капель росы, мяса и др.).

==== Основы алхимии ====
  * **Котел** (Alchemy Pot) - предмет для смешивания ингредиентов, находится в каждом городе
  * **Рецепты** - фиксированные комбинации ингредиентов для создания конкретных зелий
  * **Эксперименты** - можно пытаться смешивать неизвестные комбинации (рискованно)

==== Основные ингредиенты ====
  * **[[ru:rpd:dewdrop_item|Капля росы]]** - базовый реагент для большинства зелий
  * **Семена растений** - определяют тип создаваемого зелья
  * **Мясо и органы мобов** - для специальных рецептов
  * **Пыль и эссенции** - редкие ингредиенты для мощных зелий

==== Известные рецепты ====
  * **2x Семя Солнечника + Капля росы** = [[ru:rpd:potion_of_healing_item|Зелье исцеления]]
  * **Семя Огнецвета + Таинственное мясо** = Зелье жидкого огня
  * **Семя Ледяника + Капля росы** = [[ru:rpd:potion_of_frost_item|Зелье мороза]]
  * **Семя Сновидца + Капля росы** = Зелье левитации
  * **Трупная пыль + Капля росы + Осколок кости** = Зелье силы + Скелет

==== Механика ====
  * Ингредиенты кладутся в котел один за другим
  * При правильной комбинации создается заданное зелье
  * При неправильной - может получиться случайное зелье или взрыв
  * Некоторые рецепты дают несколько результатов

==== Стратегия ====
  * Собирайте семена всех типов для гибкости в крафте
  * Капли росы - самый ценный ресурс, используйте мудро
  * Изучайте рецепты через пробу или читая соответствующие свитки
  * Зелья исцеления всегда приоритетны на ранних этапах
