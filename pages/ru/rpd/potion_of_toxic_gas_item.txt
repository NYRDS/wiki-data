====== Зелье Токсичного Газа ======

{{ rpd:images:potion_of_toxic_gas_item.png|Зелье Токсичного Газа }}

**Зелье Токсичного Газа** — взрывоопасное зелье, которое при разбивании высвобождает облако токсичного газа.

==== Описание ====
Малейшее повреждение или попытка открыть этот сосуд приведёт к взрыву закачанного под давлением токсичного газа. Тебе лучше метнуть его во врага, нежли пытаться открыть.

==== Свойства ====
  * **Тип**: Зелье
  * **Цена**: 40 золота
  * **Вес**: 1

==== Эффект ====
  * При разбивании создает облако токсичного газа
  * Наносит урон всем существам в облаке
  * Газ сохраняется на карте и продолжает наносить урон

==== Использование ====
  * Бросьте в точку на карте, чтобы создать облако токсичного газа
  * Используется для отравления врагов
  * Может быть использовано для создания укрытия или для побега

==== Создание ====
  * Можно получить при приготовлении [[ru:rpd:sorrowmoss_plant|Растения Горя]] в алхимическом котле

==== Применение ====
  * Используется для создания [[ru:rpd:poison_arrow_item|Ядовитых стрел]] путем смачивания обычных стрел
