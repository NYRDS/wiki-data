====== Роба Некроманта ======

{{ rpd:images:necromancer_robe_item.png|Роба Некроманта }}

**Роба Некроманта** — это классовая броня в Remixed Pixel Dungeon для класса [[ru:rpd_necromancer_class|Некромант]].

==== Описание ====
**Роба Некроманта** — это начальная броня для класса [[ru:rpd_necromancer_class|Некромант]]. Она обеспечивает уникальные некромантские способности, которые усиливают основной геймплей некроманта, связанный с собиранием и использованием душ.

==== Характеристики ====
  * **Класс Брони**: Тканевая броня
  * **Прочность**: Стандартная для тканевой брони
  * **Специальная способность**: "Некромантия" — потребляет души для выполнения некромантских действий
  * **Сбор душ**: Даёт 1 очко души за каждого убитого монстра

==== Особые свойства ====
  * **Сбор душ**: Автоматически даёт 1 душу каждый раз, когда владелец побеждает монстра
  * **Некромантские способности**: Позволяет использовать некромантские способности, потребляющие души
  * **Только для класса**: Только [[ru:rpd_necromancer_class|Некромант]] может эффективно носить эту броню
  * **Призыв существ**: Позволяет использовать потребляющие души способности для призыва нежити, например [[ru:rpd_deathling_mob|Смертенышей]]

==== Механика ====
  * Функционирует как основная часть геймплея [[ru:rpd_necromancer_class|Некроманта]], связанного с собиранием душ через боевые действия и их использованием в способностях
  * Сбор душ является основой для системы ресурса [[ru:rpd_souls|Души]]
  * Работает в сочетании с другими некромантскими предметами и [[ru:rpd_necromancy_spell|Заклинанием Некромантии]]

==== Классы ====
  * [[ru:rpd_necromancer_class|Некромант]] — Предназначено специально для этого класса

==== Стратегия ====
  * Необходима для максимальной эффективности класса [[ru:rpd_necromancer_class|Некромант]]
  * Сосредоточьтесь на бою для эффективного сбора душ
  * Балансируйте между атакующими и защитными некромантскими способностями
  * Подумайте о повышении возможностей генерации душ
