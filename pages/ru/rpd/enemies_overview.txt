====== Враги ======

==== Описание ====
Враги - это враждебные мобы в Remixed Pixel Dungeon, которые атакуют игрока при виде. Они являются основным источником боевых испытаний в игре.

==== Категории ====
  * **Крысы** - Основные враги, встречающиеся по всему подземелью
  * **Гноллы** - Гуманоидные враги с различными способностями
  * **Крабы** - Враги с высокой защитой в стоках
  * **Пауки** - Паукообразные враги, способные ловить игроков в сети
  * **Элементали** - Враги с магической тематикой, основанные на элементах
  * **Нежить** - Воскрешенные трупы с особыми способностями
  * **Мимики** - Сундуки с сокровищами, которые атакуют при открытии
  * **Боссы** - Мощные враги с уникальными механиками

==== Общие свойства врагов ====
  * **Враждебность** - Враги атакуют игрока при прямой видимости
  * **Диапазон агрессии** - Большинство врагов будут преследовать игрока на короткое расстояние
  * **Добыча** - Враги дают лут при поражении
  * **Поведение ИИ** - У врагов разные паттерны интеллекта (от простых до сложных)

==== Боевые взаимодействия ====
  * Враги будут атаковать, если игрок находится рядом
  * У некоторых врагов есть атаки дальнего боя
  * У врагов могут быть специальные способности или иммунитеты
  * Разные типы врагов могут появляться на разных уровнях подземелья

==== Распределение по уровням подземелья ====
  * **Стоки** - Крысы, Гноллы, Крабы, Слизни
  * **Тюрьма** - Скелеты, Охранники, Призраки
  * **Пещеры** - Летучие мыши, Монстры, Пауки, Шаман
  * **Метрополис** - Элементали, Големы, Чернокнижники, Монахи
  * **Покои** - Демоны, Суккубы, Рыцари смерти, Слуги Йога
