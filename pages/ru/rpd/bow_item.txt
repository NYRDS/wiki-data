====== Луки ======

**Луки** — категория [[ru:rpd:weapon|оружия]] дальнего боя в Remixed Dungeon, использующая [[ru:rpd:arrow|стрелы]] как боеприпасы.

===== Общая информация =====
  * **Тип:** [[ru:rpd:weapon|Оружие]] дальнего боя
  * **Слот:** Основная рука
  * **Дальность:** 5 клеток
  * **Боеприпасы:** [[ru:rpd:arrow|Стрелы]] (требуются для атаки)

===== Основные характеристики =====
  * [[ru:rpd:damage|Урон]] — влияет на наносимый урон при попадании
  * [[ru:rpd:speed|Скорость]] — определяет частоту атак
  * [[ru:rpd:accuracy|Точность]] — шанс попадания по цели
  * [[ru:rpd:tier|Уровень]] — влияет на характеристики и улучшения

===== Известные луки =====
  * [[ru:rpd:wooden_bow_item|Деревянный лук]] — начальный лук
  * [[ru:rpd:long_bow_item|Длинный лук]] — улучшенная версия
  * [[ru:rpd:hand_crossbow_item|Ручной арбалет]] — быстрый арбалет
  * [[ru:rpd:crossbow_item|Арбалет]] — сильный арбалет
  * [[ru:rpd:rune_crossbow_item|Рунический арбалет]] — магический арбалет
  * [[ru:rpd:rune_bow_item|Рунический лук]] — магический лук
  * [[ru:rpd:ruby_bow_item|Рубиновый лук]] — мощный лук для экспертов
  * [[ru:rpd:ceramic_crossbow_item|Керамический арбалет]] — уникальный арбалет

===== Особенности =====
  * Требуют [[ru:rpd:arrow|стрелы]] для атаки (расходуются при использовании)
  * Имеют ограниченную дальность действия
  * Позволяют атаковать на расстоянии
  * Могут быть улучшены с помощью [[ru:rpd:scroll_of_upgrade_item|свитка улучшения]]

===== Стратегия =====
  * Полезны для атаки с безопасного расстояния
  * Эффективны против одиночных целей
  * Зависят от запаса [[ru:rpd:arrow|стрел]]
  * Требуют точного управления дистанцией до цели
