====== Identificación ======

**Identificación** es el proceso de descubrir las propiedades y efectos de objetos no identificados en Remixed Dungeon.

==== Métodos de Identificación ====
  * **Pergamino de Identificación**: El método principal para identificar objetos
  * **Usar/Equipar Objetos**: El equipo se identifica gradualmente a medida que se usa
  * **Poción de Experiencia**: Puede identificar equipo cuando se usa
  * **Pozo de Conciencia**: Una característica de nivel que puede identificar objetos
  * **Anillo de Detección**: Mejora la posibilidad de identificar objetos al usarlos

==== Objetos Identificados ====
  * **Estadísticas Completas**: Muestra estadísticas y propiedades completas del objeto
  * **Seguro de Usar**: Los objetos malditos identificados muestran estado de "maldición"
  * **Efectos de Poción/Pergamino**: Revela efecto exacto de consumibles
  * **Encantamientos/Glifos**: Muestra encantamiento o glifo específico en el equipo

==== Objetos No Identificados ====
  * **Información Limitada**: Solo se ve información básica
  * **Riesgo de Maldiciones**: Objetos malditos pueden tener efectos negativos al usarlos
  * **Valor Desconocido**: Pueden ser más o menos valiosos de lo asumido

==== Consejos de Identificación ====
  * **Priorizar Pociones**: Identificar pociones de curación primero para asegurar seguridad
  * **Armas/Armadura**: Identificar equipo antes de mejorar para saber si está maldito
  * **Anillos**: Identificar anillos temprano para determinar su efecto
  * **Reemplazo de Pergamino**: Mantener pergamino de identificación disponible para objetos importantes
