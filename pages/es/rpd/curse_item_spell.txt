====== Hechizo de Maldición de Objeto ======

{{ rpd:images:curse_item_spell_icon.png|Curse Item Spell Icon }}

**Hechizo de Maldición de Objeto** es un hechizo en Remixed Pixel Dungeon (Afinidad con la Necromancia).

==== Descripción ====
El Hechizo de Maldición de Objeto permite al héroe seleccionar un objeto de su mochila y maldecirlo. Los objetos malditos se vuelven permanentes al héroe y no se pueden quitar hasta que se limpie la maldición, pero muchos también obtienen propiedades mejoradas.

==== Estadísticas ====
  * **Afinidad Mágica:** [[es:rpd:necromancy|Necromancia]]
  * **Objetivo:** Uno mismo
  * **Nivel:** 1
  * **Coste de Maná:** 3 (coste del hechizo)
  * **Tiempo de Reutilización:** 5 turnos
  * **Tiempo de Lanzamiento:** 0 turnos
  * **Efectos Especiales:** Maldice un objeto seleccionado del inventario

==== Uso ====
El Hechizo de Maldición de Objeto se puede usar para:
  * Maldecir armas para potencialmente obtener propiedades mejoradas
  * Maldecir armaduras para potencialmente obtener propiedades mejoradas
  * Deliberadamente vincular objetos que quieres asegurar que no pierdas accidentalmente
  * Mejora estratégica de equipamiento a través de las mecánicas de maldición

==== Clases ====
Las clases que pueden usar este hechizo incluyen:
  * Clases con afinidad de Necromancia
  * Otras clases que obtengan acceso por medios especiales

==== Estrategia ====
Cómo usar eficazmente el hechizo de Maldición de Objeto:
  * La maldición a veces puede mejorar el rendimiento del objeto (por ejemplo, las armas pueden volverse encantadas, las armaduras pueden ganar glifos)
  * Ten en cuenta que los objetos malditos no pueden ser desequipados hasta que sean bendecidos con una [[es:rpd:potion_of_holy_water|Poción de Agua Bendita]] o medios similares
  * Considera la compensación entre posibles beneficios e imposibilidad de quitar el objeto maldito
  * Útil para asegurar que el equipo clave permanezca equipado durante situaciones peligrosas
  * Usar con cuidado con objetos consumibles ya que podrían no funcionar como se espera cuando están malditos
