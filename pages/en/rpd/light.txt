====== Light ======

{{ rpd:images:buff_Light.png|Light Buff Icon }}

The **Light** buff is a status effect in Remixed Pixel Dungeon that provides illumination for the hero character.

==== Description ====
The Light buff provides illumination that lights up the surrounding area, making it easier to see in dark dungeon levels. This helps reveal hidden traps, doors, and items that might otherwise be missed in the dark.

==== Effects ====
  * **Illumination:** Lights up the area around the hero
  * **Duration:** Varies by source, typically 80 turns for Magic Torch spell
  * **Trap Detection:** Makes it easier to spot traps that require light to see
  * **Secret Discovery:** Helps identify secret doors and hidden features

==== Sources ====
The Light buff can be obtained from:
  * [[en:rpd:magic_torch_spell|Magic Torch]] spell - Provides 80 turns of light
  * [[en:rpd:torch|Torch]] item - Provides light for a limited time when carried
  * Potentially other sources in the game

==== Strategy ====
  * Use before exploring dark areas to avoid surprises
  * Particularly beneficial in the Prison and Halls levels
  * Helps with navigation and item identification in dark sections
  * Can be essential for finding secret doors and hidden areas
