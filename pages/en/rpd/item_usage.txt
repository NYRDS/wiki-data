====== Item Usage ======

This page explains how to use items in Remixed Dungeon.

==== Basic Item Usage ====
Items in Remixed Dungeon can be used in multiple ways:

==== From Inventory ====
  * Open your [[en:rpd_inventory|inventory]] by pressing the inventory button
  * Select an item from the list
  * Press the "Use" button to use the item
  * Some items may have additional options (e.g. "Equip" for weapons and armor)

==== From Quickslots ====
  * Items can be assigned to [[en:rpd_quickslots|quickslots]] for fast access
  * Press the corresponding number key (1-9, 0) to use the item directly
  * No need to open inventory for quickslot items

==== Special Item Types ====
  * [[en:rpd_potions|Potions]]: Quaff to apply effect
  * [[en:rpd_scrolls|Scrolls]]: Read to cast spell or effect
  * [[en:rpd_food|Food]]: Eat to restore health or satiety
  * [[en:rpd_weapons|Weapons]]: Equip to use in combat
  * [[en:rpd_armor|Armor]]: Equip to increase defense

==== Item Restrictions ====
Some items have usage restrictions:
  * Items requiring minimum [[en:rpd_strength|strength]] cannot be equipped if the hero is too weak
  * [[en:rpd_cursed_items|Cursed items]] may malfunction when used
  * Some items have limited charges and must be recharged
