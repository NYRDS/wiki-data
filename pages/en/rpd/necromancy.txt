====== Necromancy ======

**Necromancy** is a magical discipline in Remixed Pixel Dungeon related to death, undead creatures, and corpse manipulation. It represents one of the game's magic affinities.

==== Overview ====
Necromancy encompasses various mechanics, items, enemies, and gameplay elements related to death and undead themes in the dungeon. It is one of the magic affinities in the game, alongside [[en:rpd:elemental_mob|Elemental]], Combat, Rogue, Huntress, [[en:rpd:elf_class|Elf]], Witchcraft, Priest, and PlagueDoctor.

==== Classes ====
  * [[en:rpd:necromancer_class|Necromancer]] - The primary necromancy-focused class
  * [[en:rpd:lich_subclass|Lich]] - A subclass for the Necromancer (obtained via [[en:rpd:black_skull_of_mastery_item|Black Skull of Mastery]])

==== Locations ====
  * [[en:rpd:necropolis|Necropolis]] - The necromancy-themed branch dungeon, accessed via [[en:rpd:sewers|Sewers]] or [[en:rpd:prison|Prison]] with a [[en:rpd:skeleton_key_item|Skeleton Key]]

==== Items ====
  * [[en:rpd:sacrificial_sword_item|Sacrificial Sword]] - A necromantic weapon
  * [[en:rpd:corpse_dust_item|Corpse Dust]] - Item obtained from skeleton remains
  * [[en:rpd:black_skull_item|Black Skull of Mastery]] - Necromancer subclass artifact for raising minions
  * [[en:rpd:necromancer_robe_item|Necromancer Robe]] - Starting armor for the Necromancer class
  * [[en:rpd:necromancer_armor_item|Necromancer Armor]] - Class armor obtained using [[en:rpd:armor_kit_item|Armor Kit]]

  * [[en:rpd:spell_book_item|Spell Book]] - Can contain necromantic spells

==== Mechanics ====
  * [[en:rpd:necrotism|Necrotism]] - A spreading death effect that can cause game over
  * [[en:rpd:summon_deathling_spell|Summon Deathling]] - Create undead minions (Necromancy affinity spell)
  * Corpse and skeleton interactions using [[en:rpd:carcass|Carcass]] items
  * [[en:rpd:raise_dead_spell|Raise Dead]] - Reanimate defeated enemies (Necromancy affinity spell)
  * [[en:rpd:exhumation_spell|Exhumation]] - Raise the dead from graves (Necromancy affinity spell)
  * [[en:rpd:possess_spell|Possess]] - Control enemy bodies (Necromancy affinity spell)
  * [[en:rpd:dark_sacrifice_spell|Dark Sacrifice]] - Sacrificial magic (Necromancy affinity spell)

==== Spells ====
All necromancy spells have the Necromancy magic affinity:
  * [[en:rpd:summon_deathling_spell|Summon Deathling]] - Summon undead minion
  * [[en:rpd:raise_dead_spell|Raise Dead]] - Reanimate fallen enemies
  * [[en:rpd:exhumation_spell|Exhumation]] - Raise the dead from graves
  * [[en:rpd:possess_spell|Possess]] - Control enemy bodies
  * [[en:rpd:dark_sacrifice_spell|Dark Sacrifice]] - Sacrificial magic

==== Strategy ====
Playing with necromancy mechanics involves managing undead minions, using death-based effects strategically, and taking advantage of the unique abilities that come with necromantic themes. The [[en:rpd:necromancer_class|Necromancer]] class excels at raising undead minions to fight alongside them, and their [[en:rpd:necromancer_robe_item|Necromancer Robe]] provides bonuses when fighting undead enemies.
