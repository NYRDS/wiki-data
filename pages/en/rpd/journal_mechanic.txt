====== Journal ======

The Journal is a gameplay feature in Remixed Dungeon that helps players track important information, locations, and discoveries in the dungeon.

==== Description ====
The Journal is an in-game tool that records information about important locations, NPCs, items, and other discoveries made during a dungeon run. It helps players keep track of significant elements they've encountered.

==== Function ====
  * **Information Tracking:** Records important locations and discoveries
  * **NPC Recognition:** Tracks NPCs the player has encountered
  * **Location Marking:** Marks special locations like shops, altars, etc.
  * **Reference Tool:** Allows players to recall important information

==== Journal Entries ====
The journal typically records:
  * Special locations (shops, altars, fountains, etc.)
  * Important NPCs (Ghost, Dwarf, etc.)
  * Notable items or equipment
  * Special dungeon features

==== Interface ====
  * Accessible from the game interface
  * Provides quick reference to important locations
  * Helps with navigation and memory
