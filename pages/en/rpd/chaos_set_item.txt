====== Chaos Set ======

{{ rpd:images:item_ChaosArmor.png|Chaos Set }}

==== Description ====
The **Chaos Set** refers to a collection of items created using [[en:rpd:chaos_crystal_item|Chaos Crystals]]. This set includes chaotic versions of various equipment types.

==== Components ====
The Chaos Set includes:
  * [[en:rpd:chaos_sword_item|Chaos Sword]]
  * [[en:rpd:chaos_bow_item|Chaos Bow]]
  * [[en:rpd:chaos_staff_item|Chaos Staff]]
  * [[en:rpd:chaos_armor_item|Chaos Armor]]
  * [[en:rpd:chaos_shield_item|Chaos Shield]]

==== Special Properties ====
  * Each piece is created by fusing a [[en:rpd:chaos_crystal_item|Chaos Crystal]] with the base item
  * All items in the set contain chaotic energy
  * Individual pieces have unique mechanics (see individual pages)
  * Powerful but potentially unpredictable effects
  * Chaos Sword has special damage and upgrade mechanics based on combat damage dealt

==== Creation ====
To create Chaos Set items, you need to fuse a [[en:rpd:chaos_crystal_item|Chaos Crystal]] with the corresponding base items:

  * [[en:rpd:chaos_sword_item|Chaos Sword]]: Fuse Chaos Crystal with any melee weapon (requires 50+ charge on crystal)
  * [[en:rpd:chaos_bow_item|Chaos Bow]]: Fuse Chaos Crystal with any bow (requires 50+ charge on crystal)
  * [[en:rpd:chaos_staff_item|Chaos Staff]]: Fuse Chaos Crystal with any wand (requires 50+ charge on crystal)
  * [[en:rpd:chaos_armor_item|Chaos Armor]]: Fuse Chaos Crystal with any armor (requires 50+ charge on crystal)
  * [[en:rpd:chaos_shield_item|Chaos Shield]]: Fuse Chaos Crystal with any shield (requires 50+ charge on crystal)

==== Strategy ====
Items in the chaos set offer unique gameplay mechanics with self-upgrading abilities, making them powerful but requiring active use to maintain their effectiveness. The Chaos Crystal must have at least 50 charges to create any Chaos item, and identifying the crystal requires 2 steps of identification.
