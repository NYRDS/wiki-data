====== Rogue Armor ======

{{ rpd:images:rogue_armor_item.png|Rogue Armor }}

**Rogue Armor** is a class-specific armor for the [[en:rpd:rogue_class|Rogue]] class in Remixed Dungeon. It comes with the special ability "Smoke Bomb".

==== Statistics ====
  * **Type:** Class Armor
  * **Required Class:** [[en:rpd:rogue_class|Rogue]]
  * **Required Subclass:** Requires completing a mastery challenge (e.g., [[en:rpd:freerunner_subclass|Freerunner]])
  * **Armor Class:** Medium
  * **Durability:** Medium
  * **Stealth Bonus:** High

==== Abilities ====
  * **Smoke Bomb:** Creates a smoke screen that allows the rogue to become invisible for a short time and teleport to any visible location
  * **Enhanced Stealth:** Improved stealth capabilities compared to regular armor
  * **Mobility:** Allows for quick escapes and repositioning during combat

==== Description ====
Rogue Armor is designed for rogues who have mastered their stealth and mobility skills. Its Smoke Bomb ability provides exceptional escape options and tactical positioning advantages, making it ideal for hit-and-run tactics.

==== Acquisition ====
To obtain Rogue Armor:

  1. Find an [[en:rpd:armor_kit_item|Armor Kit]] from the [[en:rpd:king_mob|King]] boss in the Halls level

  2. Upgrade a regular armor piece using the Armor Kit

  3. Select the "Rogue Armor" specialization when prompted

==== Special Effects ====
  * **Invisibility:** Temporarily renders the rogue invisible when using Smoke Bomb
  * **Teleportation:** Allows the rogue to teleport to any visible location during invisibility
  * **Tactical Escape:** Provides multiple options for escaping difficult situations
