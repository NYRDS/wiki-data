====== Class Armor ======

==== Description ====
**Class Armor** refers to the unique, powerful armor sets available to each hero class in Remixed Dungeon. These armors provide special abilities and bonuses in addition to standard defensive properties. Each class has a unique class armor that unlocks at tier 2 of that class's armor tree.

==== Class-Specific Armors ====
  * [[en:rpd:warrior_class|Warrior]] → [[en:rpd:warrior_armor_item|Warrior Armor]] (base), [[en:rpd:gladiator_armor_item|Gladiator Armor]] (Gladiator subclass), [[en:rpd:berserker_armor|Berserker Armor]] (Berserker subclass)
  * [[en:rpd:mage_class|Mage]] → [[en:rpd:mage_armor_item|Mage Armor]] (base), [[en:rpd:warlock_armor_item|Warlock Armor]] (Warlock subclass), [[en:rpd:battle_mage_armor_item|Battle Mage Armor]] (Battle Mage subclass)
  * [[en:rpd:rogue_class|Rogue]] → [[en:rpd:rogue_armor_item|Rogue Armor]] (base), [[en:rpd:assassin_armor_item|Assassin Armor]] (Assassin subclass), [[en:rpd:freerunner_armor|Freerunner Armor]] (Freerunner subclass)
  * [[en:rpd:huntress_class|Huntress]] → [[en:rpd:huntress_armor_item|Huntress Armor]] (base), [[en:rpd:sniper_armor_item|Sniper Armor]] (Sniper subclass), [[en:rpd:warden_armor_item|Warden Armor]] (Warden subclass)
  * [[en:rpd:elf_class|Elf]] → [[en:rpd:elf_armor|Elf Armor]] (base), [[en:rpd:scout_armor_item|Scout Armor]] (Scout subclass), [[en:rpd:shaman_armor_item|Shaman Armor]] (Shaman subclass)
  * [[en:rpd:necromancer_class|Necromancer]] → [[en:rpd:necromancer_armor_item|Necromancer Armor]] (base and Lich subclass), [[en:rpd:witchdoctor_armor|Witchdoctor Armor]] (Witchdoctor subclass)

==== Special Abilities ====
Each class armor provides unique special abilities:
  * [[en:rpd:gladiator_armor_item|Gladiator Armor]]: Throw weapon and return it to the hero
  * [[en:rpd:assassin_armor_item|Assassin Armor]]: Blink to a targeted location
  * [[en:rpd:freerunner_armor|Freerunner Armor]]: Phase through walls and gain speed boost
  * [[en:rpd:battle_mage_armor|Battle Mage Armor]]: Empower the staff with magical energy
  * [[en:rpd:shaman_armor_item|Shaman Armor]]: Special staff-based mechanics
  * [[en:rpd:witchdoctor_armor|Witchdoctor Armor]]: Necromancy-related abilities
  * [[en:rpd:plague_doctor_armor_item|Plague Doctor Armor]]: Disease and healing mechanics
  * [[en:rpd:warden_armor_item|Warden Armor]]: Terrain manipulation abilities
  * [[en:rpd:berserker_armor|Berserker Armor]]: Rage and health conversion abilities

==== Obtaining ====
Class armor can be obtained in two ways:

  1. **Crafting**: Using an [[en:rpd:armor_kit_item|Armor Kit]] on regular armor of the same tier

  2. **Random drops**: Found as rare drops from various sources throughout the dungeon
