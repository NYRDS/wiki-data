====== User Interface ======

The user interface (UI) in Remixed Dungeon is designed to provide all necessary information to the player while keeping the game screen clean and intuitive.

==== Main Interface Components ====
  * **Hero Status**: Displays hero's [[en:rpd_hp|HP]], [[en:rpd_mana_mechanic|mana]], and [[en:rpd_level|level]]
  * [[en:rpd_quickslots|Quickslots]]: Quick access toolbar at the bottom of the screen
  * **Dungeon View**: Main view showing the dungeon and characters
  * **Control Buttons**: Movement, inventory, and other actions
  * **Info Panel**: Shows detailed information about selected elements

==== Interface Settings ====
The interface can be customized in the settings menu:
  * Adjust the size of interface elements
  * Configure [[en:rpd_hotkeys|hotkeys]]
  * Toggle visibility of certain interface elements
  * Adjust brightness and contrast

==== Navigation ====
  * **Touch Controls**: On mobile devices, tap to move, interact, or use items
  * **Keyboard Controls**: On desktop, use WASD or arrow keys to move
  * **Mouse Controls**: Click to move, right-click to interact

==== Special UI Elements ====
  * **Hero Health Bar**: Shows current health and health trend
  * **Quickslots**: [[en:rpd_hotbars|Hotbars]] for frequently used items
  * **Dungeon Map**: Shows explored areas and reveals nearby items/mobs
  * **Action Bar**: Provides quick access to common actions
