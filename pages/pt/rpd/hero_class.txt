====== Classes de Heróis ======

{{ rpd:images:warrior_hero.png|Hero Classes }}

Classes de heróis são os diferentes tipos de personagens disponíveis para jogadores em Remixed Dungeon. **Classes de Heróis** determinam o equipamento inicial, estatísticas e habilidades especiais do personagem. Cada classe tem vantagens e estilos de jogo únicos que as tornam mais adequadas para diferentes abordagens do calabouço.

==== Classes Disponíveis ====
  * [[pt:rpd:warrior_class|Guerreiro]] - Forte e resistente
  * [[pt:rpd:mage_class|Mago]] - Feitiços poderosos e varinhas
  * [[pt:rpd:necromancer_class|Necromante]] - Classe especializada em magia necromântica

==== Características das Classes ====
  * **Equipamento Inicial:** Cada classe começa com equipamento específico
  * **Habilidades Especiais:** Habilidades únicas de classe que diferenciam os estilos de jogo
  * **Afinidades Estatísticas:** Classes se destacam em áreas diferentes
  * **Estilo de Jogo:** Cada classe encoraja diferentes abordagens para exploração do calabouço

==== Detalhes das Classes ====
  * **Guerreiro:** Começa com uma arma e armadura únicas, e uma ração comestível. Tem uma habilidade especial para curar-se ao lutar contra inimigos enfraquecidos.
  * **Mago:** Começa com uma vara mágica que pode ser usada como arma ou para armazenar varinhas. Tem uma habilidade especial para recarregar itens rapidamente.
  * **Necromante:** Começa com equipamento necromântico e pode erguer mortos-vivos. Tem uma habilidade especial para controlar inimigos temporariamente.

==== Mecânicas de Classe ====
Cada classe de herói começa com:
  * Arma única
  * Armadura única
  * Habilidade especial que se ativa sob condições específicas
  * [[pt:rpd:food_item|Comida]] única

==== Subclasses ====
Jogadores avançados podem desbloquear [[pt:rpd:subclasses|subclasses]] (caminhos de maestria) após vencer com uma classe:
  * Guerreiro: [[pt:rpd:warrior_subclass|Guerreiro Subclasse]] ou [[pt:rpd:guardian_subclass|Guardião]]
  * Mago: Subclasses de Mago
  * Ladino: Subclasses de Ladino
  * Caçadora: Subclasses de Caçadora
  * Elfo: Subclasses de Elfo
  * Necromante: [[pt:rpd:lich_subclass|Lich]]

==== Dicas de Seleção de Classe ====
  * **Novos Jogadores:** Guerreiro é geralmente o mais tolerante para iniciantes
  * **Jogadores Avançados:** Cada classe oferece desafios e estratégias únicas
  * **Correspondência com Estilo de Jogo:** Escolha baseada na abordagem preferida de jogo
  * **Experimentação:** Tente diferentes classes para encontrar sua preferência
  * **Longo prazo:** Considere como as classes se desenvolvem em múltiplas partidas
