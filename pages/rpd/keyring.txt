====== Keyring ======

{{ rpd:images:item_Keyring.png|Keyring }}

Keyring is a special container for artifacts in Remixed Dungeon. Unlike regular bags, it can only hold specific items called Artifacts.

==== Description ====
The Keyring is a special bag that can only hold artifacts. It provides a dedicated storage space for these powerful magical items that would otherwise take up regular inventory slots.

==== Properties ====
  * **Capacity**: Holds a limited number of artifacts
  * **Specialization**: Only accepts artifacts, not regular items
  * **Function**: Keeps artifacts organized and accessible
