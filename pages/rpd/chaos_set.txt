====== Chaos Set ======

==== Description ====
The Chaos Set refers to a collection of items created using Chaos Crystals. This set includes chaotic versions of various equipment types.

==== Components ====
The Chaos Set typically includes:
  * [[rpd:chaos_sword|Chaos Sword]]
  * [[rpd:chaos_bow|Chaos Bow]]
  * [[rpd:chaos_staff|Chaos Staff]]
  * [[rpd:chaos_armor|Chaos Armor]]
  * [[rpd:chaos_shield|Chaos Shield]]

==== Special Properties ====
  * Each piece is created by fusing a Chaos Crystal with the base item
  * All items in the set contain chaotic energy
  * May have synergistic effects when worn together (if applicable)
  * Powerful but potentially unpredictable effects

==== Creation ====
To create Chaos Set items, you need to fuse Chaos Crystals with the corresponding base items:
  * Chaos Sword: Fuse Chaos Crystal with any weapon
  * Chaos Bow: Fuse Chaos Crystal with any bow
  * Chaos Staff: Fuse Chaos Crystal with any wand
  * Chaos Armor: Fuse Chaos Crystal with any armor
  * Chaos Shield: Fuse Chaos Crystal with any shield

==== Notes ====
The Chaos Set represents some of the most powerful equipment in the game, but with the risk of unpredictable chaotic effects.
