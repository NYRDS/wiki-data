====== Sniper Armor ======

{{ rpd:images:item_HuntressArmor.png|Sniper Armor }}

Sniper Armor is the class-specific armor for the [[rpd:sniper|Sniper]] subclass in Remixed Dungeon.

==== Description ====
Sniper Armor is obtained by using an [[rpd:armor_kit|Armor Kit]] on any regular armor while playing as a Sniper. This armor enhances the Sniper's unique combat abilities, particularly their ranged combat effectiveness.

==== Stats ====
  * **Tier:** Variable (based on base armor used)
  * **Protection:** Based on the original armor used with the Armor Kit
  * **Durability:** Based on the original armor used with the Armor Kit
  * **Special Effect:** Enhances the Sniper's ability to ignore enemy defense when using ranged weapons

==== Special Abilities ====
  * **Ranged Defense Ignore:** Allows the Sniper to ignore enemy defense when using ranged weapons
  * **Sniper Enhancement:** Provides additional bonuses specific to the Sniper subclass

==== Availability ====
  * Created by using an Armor Kit on any armor when playing as a Sniper
  * Only available to players who have unlocked the Sniper subclass

==== Strategy ====
  * Essential for maximizing the Sniper's effectiveness
  * Should be created as soon as an Armor Kit is found
  * Choose the best available armor to transform when creating Sniper Armor
