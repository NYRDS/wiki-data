====== Hero Classes ======

Hero classes in Remixed Pixel Dungeon define the starting equipment, stats, and special abilities of the playable characters. Each class has its own unique playstyle and mechanics.

==== Available Classes ====
  * [[rpd:warrior_class|Warrior]] - A melee-focused class with high durability
  * [[rpd:mage_class|Mage]] - A spellcaster with various magical abilities
  * [[rpd:rogue_class|Rogue]] - A stealth-focused class with high evasion
  * [[rpd:huntress_class|Huntress]] - A ranged combat specialist
  * [[rpd:elf_class|Elf]] - A natural magic user with ranged capabilities
  * [[rpd:necromancer_class|Necromancer]] - A death magic specialist
  * [[rpd:gnoll_class|Gnoll]] - A unique class with witchcraft affinity
  * [[rpd:priest_class|Priest]] - A healing-focused class
  * [[rpd:doctor_class|Doctor]] - A class focused on disease mechanics

==== Class Mechanics ====
Each hero class has:
  * Unique starting equipment
  * Special class abilities or affinities
  * Different base stats (strength, health, etc.)
  * Unique class armor obtained through the [[rpd:armor_kit|Armor Kit]]

==== Class Unlocking ====
  * All classes except Doctor and Priest are available from the start
  * The [[rpd:huntress_class|Huntress]], [[rpd:elf_class|Elf]], and [[rpd:gnoll_class|Gnoll]] classes can also be unlocked by achieving specific badges or donating to the developer
  * Some classes have special unlock requirements as documented on their individual pages
