====== Hero Classes ======

Hero classes in Remixed Pixel Dungeon define the starting equipment, stats, and special abilities of the playable characters. Each class has its own unique playstyle and mechanics.

==== Available Classes ====
  * [[rpd:warrior_class|Warrior]] - A melee-focused class with high durability - [[mr:warrior_class|Code Reference]]
  * [[rpd:mage_class|Mage]] - A spellcaster with various magical abilities - [[mr:mage_class|Code Reference]]
  * [[rpd:rogue_class|Rogue]] - A stealth-focused class with high evasion - [[mr:rogue_class|Code Reference]]
  * [[rpd:huntress_class|Huntress]] - A ranged combat specialist - [[mr:huntress_class|Code Reference]]
  * [[rpd:elf_class|Elf]] - A natural magic user with ranged capabilities - [[mr:elf_class|Code Reference]]
  * [[rpd:necromancer_class|Necromancer]] - A death magic specialist - [[mr:necromancer_class|Code Reference]]
  * [[rpd:gnoll_class|Gnoll]] - A unique class with witchcraft affinity - [[mr:gnoll_class|Code Reference]]
  * [[rpd:priest_class|Priest]] - A healing-focused class - [[mr:priest_class|Code Reference]]
  * [[rpd:doctor_class|Doctor]] - A class focused on disease mechanics - [[mr:doctor_class|Code Reference]]

==== Class Mechanics ====
Each hero class has:
  * Unique starting equipment based on JSON configuration - [[mr:warrior_class|Code Reference]]
  * Special class abilities or affinities - [[mr:warrior_class|Code Reference]]
  * Different base stats (strength, health, etc.) - [[mr:warrior_class|Code Reference]]
  * Unique class armor obtained through the [[rpd:armor_kit_item|Armor Kit]] - [[mr:warrior_class|Code Reference]]
  * Class-specific immunities and resistances - [[mr:warrior_class|Code Reference]]
  * Class-specific perks and bonuses - [[mr:warrior_class|Code Reference]]

==== Class Unlocking ====
  * All classes except Doctor and Priest are available from the start - [[mr:warrior_class|Code Reference]]
  * The [[rpd:huntress_class|Huntress]], [[rpd:elf_class|Elf]], and [[rpd:gnoll_class|Gnoll]] classes can also be unlocked by achieving specific badges or donating to the developer - [[mr:warrior_class|Code Reference]]
  * Some classes have special unlock requirements as documented on their individual pages
