====== Spell Book ======

{{rpd:spell_book_books.png?200|Spell Book Books}}

{{ :rpd:spell_book.png?nolink&50|}}

A Spell Book is a magical item that contains arcane knowledge and allows the reader to learn spells. It's more like a compilation of assorted pages that were sewn together, rather than an actual book.

==== Description ====
Spell Books contain pages of magical knowledge that can be read to learn spells. Each book may contain a different spell depending on its contents and the reader's magical knowledge.

==== Function ====
  * **Spell Learning:** Reading the book teaches the reader a specific spell
  * **Magical Knowledge:** Contains arcane riddles and mysteries
  * **Study Required:** Requires careful attention to fully comprehend

==== Spells Available ====
Spell Books may contain various spells, including but not limited to:
  * [[rpd:healing_spell|Healing]] - Restores health
  * [[rpd:ignite|Ignite]] - Creates fire
  * [[rpd:root_spell|Root]] - Immobilizes enemies
  * [[rpd:wind_gust|Wind Gust]] - Pushes enemies away
  * [[rpd:freeze_globe|Freeze]] - Freezes targets
  * [[rpd:magic_torch|Magic Torch]] - Provides light
  * And other advanced spells

==== Obtaining Spell Books ====
  * Found as treasure in dungeon levels
  * Available in special shops
  * As rewards for completing quests
  * Potentially dropped by magic-using enemies

==== Class Benefits ====
  * Particularly useful for [[rpd:mage|Mages]]
  * Can benefit other magic-using classes
  * Expands the player's magical abilities

==== Note ====
The exact spell learned from a Spell Book may vary and is discovered upon reading the book.

====== Spell Book ======

==== Description ====
It looks more like a compilation of assorted pages that were sewed together, than an actual book. You might be able to make something out of it if you'll pay enough attention.

==== Identified Description ====
Within these pages you've discovered a full text of a specific spell.

==== Special Properties ====
  * Contains magical spells that can be learned
  * Each Spell Book contains a different spell
  * Reading requires time and concentration
  * Can be used to learn new spells
  * Different spell books contain different magical knowledge

==== Usage ====
To use a Spell Book, select the "READ" action. Reading will take time and must be done without interruption. After reading, the hero will learn the spell contained within.

==== Notes ====
Spell Books are rare items that teach heroes new magical abilities. Each book contains a different spell, and reading them expands the hero's magical knowledge.
