====== Ловушки ======

**Ловушки** - это скрытые устройства, активирующиеся при наступлении на них и создающие различные негативные эффекты.

==== Типы ловушек ====
  * [[rpd:poison_trap_trap|Ловушка с ядом]] - Создаёт облако ядовитого газа
  * [[rpd:fire_trap_trap|Огненная ловушка]] - Вызывает взрыв огня
  * [[rpd:paralytic_gas_trap_level_object|Ловушка парализующего газа]] - Создаёт облако парализующего газа
  * [[rpd:toxic_gas_trap_level_object|Ловушка токсичного газа]] - Создаёт облако токсичного газа
  * [[rpd:alarm_trap_level_object|Сигнальная ловушка]] - Пробуждает всех мобов на уровне
  * [[rpd:lightning_trap_level_object|Ловушка молнии]] - Наносит урон электричеством
  * [[rpd:gravel_trap_level_object|Ловушка гравия]] - Создаёт шум, привлекающий мобов
  * [[rpd:frost_trap_level_object|Ловушка мороза]] - Создаёт облако замораживающего газа
  * [[rpd:confusion_trap_level_object|Ловушка дезориентации]] - Создаёт облако дезориентирующего газа
  * [[rpd:summoning_trap_level_object|Ловушка призыва]] - Призывает мобов в области
  * [[rpd:teleportation_trap_level_object|Ловушка телепортации]] - Телепортирует героя
  * [[rpd:rockfall_trap_level_object|Ловушка камнепада]] - Вызывает обвал камней
  * [[rpd:dart_trap_level_object|Ловушка дротика]] - Выпускает отравленный дротик
  * [[rpd:ward_trap_level_object|Охранная ловушка]] - Создаёт барьер, отражающий магию

==== Обнаружение ====
  * Ловушки изначально скрыты от игрока
  * [[rpd:ring_of_detection_item|Кольцо обнаружения]] помогает обнаруживать ловушки
  * [[rpd:hero|Герой]] с высоким [[rpd:searching|навыком поиска]] может обнаруживать ловушки
  * [[rpd:scroll_of_magic_mapping_item|Свиток волшебной карты]] открывает все ловушки на уровне

==== Активация ====
  * Ловушки активируются при наступлении на них
  * Некоторые [[rpd:mobs|мобы]] также активируют ловушки
  * [[rpd:hero|Герой]] может безопасно пройти рядом с ловушкой без активации

==== Проверка содержимого ====
  * Java Class: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/levels/objects/Trap.java|Trap.java]] (базовый класс)
  * Entity Kind: Various trap types implement getEntityKind() method
  * String Resources: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/res/values/strings_all.xml|strings_all.xml]] contains trap name and description strings
  * Configuration: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/assets/levelObjects/|levelObjects/]] directory contains JSON configuration for different traps
