====== Arrows ======

{{ rpd:images:item_CurareDart.png|Arrows }}

**Arrows** are ranged ammunition items in Remixed Dungeon that can be used with ranged weapons or in alchemy processes.

==== Base Arrow ====
  * [[rpd:common_arrow_item|Common Arrow]] - Basic arrow used in moistening process

==== Special Arrows ====
  * [[rpd:poison_arrow_item|Poison Arrow]] - Created by moistening with Potion of Toxic Gas
  * [[rpd:frost_arrow_item|Frost Arrow]] - Created by moistening with Potion of Frost
  * [[rpd:fire_arrow_item|Fire Arrow]] - Created by moistening with Potion of Liquid Flame
  * [[rpd:health_arrow_item|Health Arrow]] - Created by moistening with Potion of Healing
  * [[rpd:paralysis_arrow_item|Paralysis Arrow]] - Created by moistening with Potion of Paralytic Gas
  * [[rpd:amok_arrow_item|Amok Arrow]] - Created by moistening with Potion of Strength

==== Mechanics ====
  * **Moistening**: Potions can be applied to common arrows to create special arrows with enhanced effects
  * **Usage**: Special arrows have the same function as their potion counterparts but are applied to targets via ranged attacks
  * **Ammunition**: Used with ranged weapons like bows and crossbows

==== Availability ====
  * Common arrows can be found throughout the dungeon
  * Special arrows are created via the [[rpd:moisten_mechanic|moistening]] process
  * Each special arrow type has its specific tactical uses
