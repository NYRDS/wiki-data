====== Playable Class ======

==== Description ====
Playable Classes are the different character types that players can choose when starting a new game in Remixed Dungeon.

==== Available Classes ====
  * [[rpd:warrior|Warrior]] - A strong melee fighter
  * [[rpd:mage|Mage]] - A spellcaster with magical abilities
  * [[rpd:rogue|Rogue]] - A stealthy character with detection skills
  * [[rpd:huntress|Huntress]] - An expert with ranged weapons
  * [[rpd:elf|Elf]] - An agile character with magical resistance
  * [[rpd:necromancer|Necromancer]] - A dark magic specialist
  * [[rpd:gnoll_hero|Gnoll Hero]] - A strong hyena-like warrior
  * [[rpd:lich|Lich]] - An advanced undead magic user (mastery of Necromancer)
  * [[rpd:priest|Priest]] - A divine magic user
  * [[rpd:plague_doctor|Plague Doctor]] - A disease and alchemy specialist

==== Notes ====
Each class has unique starting perks, abilities, and playstyles. Players can unlock additional classes by completing specific challenges in the dungeon.
