====== Priest ======

{{ rpd:images:priest_sprite.png|Priest }}

The **Priest** is an unlockable playable class in Remixed Dungeon focused on divine magic and healing. This class is currently in development and not yet available for selection in normal gameplay.

==== Class Overview ====
  * **Magic Affinity:** Priest magic affinity focused on healing and divine magic
  * **Special Ability:** Enhanced healing and curse removal capabilities

==== Starting Stats and Equipment ====
  * **Starting HP**: 20
  * **Starting STR**: 10
  * **Starting Skill Points**: 10
  * **Starting Equipment**:
    * Cloth Armor (identified)
    * 100 Gold
    * [[rpd:rations|Iron Ration]] (from common starting items)
    * [[rpd:scroll_of_remove_curse|Scroll of Remove Curse]] (known item, no need to identify)

==== Class Perks ====
  * **Divine Magic:** Access to special divine spells and abilities
  * **Enhanced Curse Removal:** [[rpd:scroll_of_remove_curse|Scroll of Remove Curse]] is known from the beginning

==== Special Mechanics ====
  * The Priest starts with a known [[rpd:scroll_of_remove_curse|Scroll of Remove Curse]], allowing immediate access to curse removal without identification
  * As with all classes, the Priest has unique class armor: [[rpd:priest_armor|Priest Vestments]]

==== Class Abilities ====
The Priest's magic affinity gives access to specific spells:
  * **Healing spells:** Like [[rpd:heal_spell|Heal]]
  * **Calming effects:** Like [[rpd:calm_spell|Calm]]
  * **Charm abilities:** Like [[rpd:charm_spell|Charm]]
  * **Order effects:** Like [[rpd:order_spell|Order]]

==== Class Advantages ====
  * Enhanced healing capabilities compared to other classes
  * Superior curse removal abilities (starts with known [[rpd:scroll_of_remove_curse|Scroll of Remove Curse]])
  * Support-focused magic that can benefit longer dungeon runs
  * Has associated mastery badge (MASTERY_PRIEST) and class armor ([[rpd:priest_armor|Priest Vestments]])

==== Class Disadvantages ====
  * May have limited offensive capabilities compared to other classes
  * Class is not yet unlockable through normal gameplay (needs to be unlocked via donation threshold or other method)

==== Unlock Condition ====
The Priest class is implemented in the code but currently not unlockable through normal gameplay (e.g., defeating bosses). It may require donation threshold or other unlock mechanism not yet enabled in public builds.
