====== Gnoll Armor ======

{{ rpd:images:gnollarmor_sprite.png|Gnoll Armor }}

**Gnoll Armor** is a unique armor item that can only be created by a Gnoll character using an [[rpd:armor_kit|Armor Kit]] and a tier-2 or higher armor.

==== Overview ====
Gnoll Armor is a special armor that can be crafted by Gnolls using an [[rpd:armor_kit|Armor Kit]]. It retains the base properties of the original armor but gains a unique special ability based on upgrading level. This armor has a special ability that enhances the wearer's combat effectiveness in various ways.

==== Special Ability ====
  * **Effect**: The exact ability depends on the specific implementation in the game, but typically involves enhanced combat abilities
  * **Usage**: Activated manually when needed
  * **Cooldown**: The ability has a cooldown period before it can be used again

==== Creation ====
  * Requires: Any armor of tier 2 or higher + [[rpd:armor_kit|Armor Kit]]
  * Character: Only available to Gnoll characters
  * Method: Use Armor Kit while the target armor is equipped

==== Benefits ====
  * Maintains the base properties of the original armor
  * Gains a powerful special ability that synergizes with Gnoll gameplay
  * Provides excellent tactical options in combat situations

==== Strategy ====
Gnoll Armor provides unique combat utility specific to the Gnoll character. It's particularly useful for:
  * Enhancing combat effectiveness
  * Providing tactical advantages in difficult encounters
  * Utilizing the Gnoll's unique abilities optimally
