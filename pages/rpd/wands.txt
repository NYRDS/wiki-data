====== Палочки ======

**Палочки** - это магические [[rpd:items|предметы]], которые выпускают магические заряды с различными эффектами.

==== Типы палочек ====
  * [[rpd:wand_of_magic_missile_item|Палочка волшебных снарядов]] - Наносит умеренный урон (начинает с [[rpd:mage_class|Магом]])
  * [[rpd:wand_of_lightning_item|Палочка молний]] - Наносит урон электричеством
  * [[rpd:wand_of_fireblast_item|Палочка огнеплюя]] - Создает взрыв огня
  * [[rpd:wand_of_cold_snap_item|Палочка холода]] - Наносит урон холодом
  * [[rpd:wand_of_poison_item|Палочка яда]] - Создает облако яда
  * [[rpd:wand_of_disintegration_item|Палочка дезинтеграции]] - Пронзает цели и препятствия
  * [[rpd:wand_of_blink_item|Палочка скачка]] - Телепортирует героя
  * [[rpd:wand_of_teleportation_item|Палочка телепортации]] - Перемещает цель
  * [[rpd:wand_of_regeneration_item|Палочка регенерации]] - Восстанавливает здоровье
  * [[rpd:wand_of_slowness_item|Палочка замедления]] - Замедляет цели
  * [[rpd:wand_of_fear_item|Палочка страха]] - Заставляет цели убегать
  * [[rpd:wand_of_avalanche_item|Палочка камнепада]] - Вызывает обвал
  * [[rpd:wand_of_amok_item|Палочка буйства]] - Заставляет мобов атаковать всех подряд
  * [[rpd:wand_of_telekinesis_item|Палочка телекинеза]] - Притягивает предметы и отталкивает мобов
  * [[rpd:wand_of_scorching_ray_item|Палочка жгучего луча]] - Выпускает мощный огненный луч
  * [[rpd:wand_of_prismatic_light_item|Палочка призменного света]] - Наносит урон и освещает тьму

==== Заряд и перезарядка ====
  * Палочки имеют ограниченное количество зарядов
  * Заряды восстанавливаются с течением времени
  * [[rpd:scroll_of_recharging_item|Свиток перезарядки]] мгновенно восстанавливает заряды
  * [[rpd:ring_of_power_item|Кольцо власти]] увеличивает эффективность палочек

==== Улучшение палочек ====
  * Палочки можно улучшать с помощью [[rpd:scroll_of_upgrade_item|Свитка улучшения]]
  * Уровни улучшения увеличивают урон и эффективность
  * [[rpd:wand_of_magic_missile_item|Палочку волшебных снарядов]] можно развоплотить для улучшения других палочек

==== Использование ====
  * Палочки используются с помощью действия "ВЗМАХНУТЬ"
  * Использование палочки требует один [[rpd:turns|ход]]
  * Направление атаки выбирается игроком
  * Палочки можно использовать как оружие ближнего боя

==== Проверка содержимого ====
  * Java Class: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/items/wands/Wand.java|Wand.java]] (базовый класс)
  * Entity Kind: Various wand types implement getEntityKind() method
  * String Resources: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/res/values/strings_all.xml|strings_all.xml]] contains wand name and description strings
  * Configuration: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/assets/itemsDesc/|itemsDesc/]] directory contains JSON configuration for different wands
