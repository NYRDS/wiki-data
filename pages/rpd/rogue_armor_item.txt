====== Rogue Armor ======

{{ rpd:images:rogue_armor_item.png|Rogue Armor }}

**Rogue Armor** is the [[rpd:class_armor|class armor]] for the [[rpd:rogue_class|Rogue]].

==== Description ====
Rogue Armor is a special armor obtained by using an [[rpd:armor_kit_item|Armor Kit]] on a tier-3 armor (Plate Mail, Scale Armor, or Leather Armor). It grants the wearer special stealth abilities.

==== Special Ability ====
**Phase Shift:** The Rogue can teleport to any visible location on the current level, but will become [[rpd:vertigo_buff|vertigo]]'d for 5 turns after doing so.

==== Stats ====
  * **Armor Class:** Based on the base armor used to create it
  * **Durability:** Standard for class armor
  * **Special:** Can use Phase Shift ability

==== Strategy ====
  * Use Phase Shift to escape dangerous situations
  * Use Phase Shift to get to distant items or areas quickly
  * Be aware of the vertigo debuff after using Phase Shift
  * The teleportation ability can be used strategically for positioning
