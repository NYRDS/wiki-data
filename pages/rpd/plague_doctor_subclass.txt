====== Plague Doctor Subclass (Fictional Content) ======

{{ rpd:images:doctor_hero.png|Doctor Class }}

==== Important Note ====
**This page documents fictional content.** There is no "Plague Doctor" subclass in Remixed Dungeon. This page exists to document an incorrect assumption about the game mechanics.

==== Actual Game Mechanics ====
The **Doctor** hero class exists in Remixed Dungeon, but has no associated subclasses. The Doctor class has a related quest known as the "Doctor Quest" which involves the [[rpd:plague_doctor_mob|Plague Doctor NPC]].

==== Correct Subclasses in the Game ====
The actual hero subclasses in Remixed Dungeon include:
  * **Warrior**: [[rpd:gladiator_subclass|Gladiator]] and [[rpd:berserker_subclass|Berserker]]
  * **Mage**: [[rpd:warlock_subclass|Warlock]] and [[rpd:battle_mage_subclass|Battle Mage]]
  * **Rogue**: [[rpd:assassin_subclass|Assassin]] and [[rpd:freerunner_subclass|Freerunner]]
  * **Huntress**: [[rpd:sniper_subclass|Sniper]] and [[rpd:warden_subclass|Warden]]
  * **Elf**: [[rpd:scout_subclass|Scout]] and [[rpd:shaman_subclass|Shaman]]
  * **Necromancer**: [[rpd:lich_subclass|Lich]]
  * **Gnoll**: [[rpd:guardian_subclass|Guardian]] and [[rpd:witchdoctor_subclass|Witchdoctor]]
