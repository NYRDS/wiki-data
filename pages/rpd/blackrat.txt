====== Black Rat ======

{{ rpd:images:blackrat_sprite.png|Black Rat }}

==== Description ====
Just a common house rat, not marsupial. The Black Rat is a custom mob implemented via JSON configuration to demonstrate mob creation capabilities for modders. It is functionally similar to other rat variants but with distinct appearance and properties.

==== Stats ====
  * **HP:** 5
  * **HT:** 5
  * **Defense Skill:** 1
  * **Attack Skill:** 10
  * **Damage:** 1-1
  * **Armor:** 1 (DR)
  * **EXP:** 1
  * **Max Level:** 1
  * **Speed:** 2x normal speed

==== Behavior ====
  * Very fast movement speed (2x normal)
  * Low attack and defense capabilities
  * Drops Rat Hide as loot

==== Implementation ====
The Black Rat demonstrates custom mob creation through JSON configuration files:
  * Defined in: `mobsDesc/BlackRat.json`
  * Uses sprite: `spritesDesc/BlackRat.json` (uses rat.png texture)
  * Scripted behavior: `scripts/mobs/ScriptedThief`

==== Usage ====
  * Primarily used as a demonstration for modders creating custom mobs
  * May appear in test levels and mod demonstrations
