====== Свитки ======

**Свитки** - это расходуемые [[rpd:items|предметы]], которые при использовании производят магические эффекты.

==== Типы свитков ====
  * [[rpd:scroll_of_identify_item|Свиток опознания]] - Опознаёт неизвестные предметы
  * [[rpd:scroll_of_magic_mapping_item|Свиток волшебной карты]] - Открывает карту уровня
  * [[rpd:scroll_of_teleportation_item|Свиток телепортации]] - Перемещает героя на уровень
  * [[rpd:scroll_of_recharging_item|Свиток перезарядки]] - Восстанавливает заряды палочек
  * [[rpd:scroll_of_remove_curse_item|Свиток снятия проклятия]] - Снимает проклятие с предметов
  * [[rpd:scroll_of_upgrade_item|Свиток улучшения]] - Улучшает предмет
  * [[rpd:scroll_of_weapon_upgrade_item|Свиток улучшения оружия]] - Улучшает оружие
  * [[rpd:scroll_of_enchantment_item|Свиток зачарования]] - Зачаровывает оружие или броню
  * [[rpd:scroll_of_terror_item|Свиток ужаса]] - Заставляет врагов убегать
  * [[rpd:scroll_of_amok_item|Свиток буйства]] - Приводит врагов в бешенство
  * [[rpd:scroll_of_psionic_blast_item|Свиток ментального удара]] - Массовая атака разума
  * [[rpd:scroll_of_lullaby_item|Свиток колыбельной]] - Усыпляет врагов
  * [[rpd:scroll_of_mirror_image_item|Свиток двойника]] - Создаёт копии героя
  * [[rpd:scroll_of_curse_item|Свиток проклятия]] - Проклинает предметы или врагов
  * [[rpd:scroll_of_domination_item|Свиток подчинения]] - Временно подчиняет врага
  * [[rpd:scroll_of_fool_item|Свиток дурака]] - Печально известен среди игроков

==== Использование ====
  * Свитки можно использовать в бою или в безопасной ситуации
  * Использование свитка требует один [[rpd:turns|ход]]
  * Многие свитки имеют область действия, показываемую при использовании
  * Некоторые свитки могут быть опознаны для постоянного использования

==== Редкость ====
  * Каждый свиток имеет свою редкость и частоту появления
  * На более глубоких [[rpd:levels|уровнях]] могут встречаться более редкие свитки
  * Свитки можно найти в [[rpd:loot|добыче]] от мобов или в сундуках

==== Проверка содержимого ====
  * Java Class: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/items/scrolls/Scroll.java|Scroll.java]] (базовый класс)
  * Entity Kind: Various scroll types implement getEntityKind() method
  * String Resources: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/res/values/strings_all.xml|strings_all.xml]] contains scroll name and description strings
  * Configuration: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/assets/itemsDesc/|itemsDesc/]] directory contains JSON configuration for different scrolls
