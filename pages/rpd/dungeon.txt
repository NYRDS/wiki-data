====== Dungeon Layout ======

The Remixed Dungeon has a complex layout with multiple interconnected levels and special areas. The dungeon structure is defined by a graph in the Dungeon.json file.

==== Main Dungeon Progression ====

The main dungeon progression follows this order:

  * [[rpd:town|Town]] (Level town_2) - Starting area with shops and services (depth 0)
  * [[rpd:sewers|Sewers]] (Levels 1-4) - Early game area with basic enemies (depth 1-4)
  * [[rpd:sewers|Sewer Boss Level]] (Level 5) - Features the Goo boss (depth 5) - see [[rpd:goo|Goo]]
  * Portal Room (Level portal_room_lvl5) - Transitional area connecting to special levels
  * [[rpd:prison|Prison]] (Levels 6-9) - Mid-game area, features Tengu boss (depth 6-9)
  * [[rpd:prison|Prison Boss Level]] (Level 10) - Features the Tengu boss (depth 10) - see [[rpd:tengu_mob|Tengu]]
  * Portal Room (Level portal_room_lvl10) - Transitional area
  * [[rpd:caves|Caves]] (Levels 11-14) - Underground caves, features DM-300 boss (depth 11-14)
  * [[rpd:caves|Caves Boss Level]] (Level 15) - Features the DM-300 boss (depth 15) - see [[rpd:dm300|DM300]]
  * Portal Room (Level portal_room_lvl15) - Transitional area
  * [[rpd:city|City]] (Levels 16-19) - Dwarven metropolis, features Dwarven King boss (depth 16-19)
  * [[rpd:city|City Boss Level]] (Level 20) - Features the King boss (depth 20) - see [[rpd:king|King]]
  * Portal Room (Level portal_room_lvl20) - Transitional area
  * [[rpd:city|Last Shop Level]] (Level 21) - Final shop before demon halls (depth 21) - see [[rpd:shops|Shops]]
  * [[rpd:halls|Halls]] (Levels 22-24) - Demon halls, features Yog-Dzewa's guardians (depth 22-24)
  * [[rpd:halls|Shadow Lord Level]] (Level 25) - Features Shadow Lord boss (depth 25) - see [[rpd:shadow_lord|Shadow Lord]]
  * Portal Room (Level portal_room_lvl25) - Transitional area
  * [[rpd:halls|Fake Last Level]] (Level 26) - Guts level entrance (depth 26) - see [[rpd:guts|Guts]]
  * [[rpd:guts|Guts]] (Levels 27-30) - Additional challenging areas (depth 27-30)
  * [[rpd:halls|Halls Boss Level]] (Level 31) - Features the final Yog-Dzewa boss (depth 31) - see [[rpd:yog|Yog-Dzewa]]
  * Portal Room (Level portal_room_lvl31) - Transitional area to final level
  * [[rpd:halls|Last Level]] (Level 32) - True final level (depth 32) - see [[rpd:yog|Yog-Dzewa]]

==== Special Areas ====

The dungeon also contains special side areas that branch from the main progression:

  * [[rpd:spider_nest|Spider Nests]] (Levels 6s-10s) - Spider-themed areas branching from Level 5 (depth 6-10)
  * [[rpd:necropolis|Necropolis]] (Levels necro1-necro5) - Undead-themed area branching from Level 7 (depth 8-12)
  * [[rpd:ice_caves|Ice Caves]] (Levels ice1-ice5) - Cold-themed area branching from Level 14 (depth 15-19)
  * [[rpd:sub_levels|Sub Levels]] - Various special themed areas

==== Town Features ====

The [[rpd:town|Town]] area (Level town_2) contains several special buildings:

  * Church (Level church_2) - Healing and blessings
  * Cinema (Level cinema_2) - Special area
  * Inn (Level inn_2) - Rest and recovery
  * Library (Level library_2) - Knowledge and information
  * [[rpd:fortunetellernpc|Fortune Teller]] (Level fortune_2) - Special services
  * [[rpd:shops|Shop]] (Level shop_town_2) - Equipment and supplies

==== Dungeon Structure Details ====

  * **Graph-Based Navigation**: Dungeon connections are defined in a graph structure in Dungeon.json
  * **Depth Values**: Each level has a depth value that determines item generation and mob stats
  * **Portal Rooms**: Special transitional levels connect main dungeon sections
  * **Branching Areas**: Special levels like Spider Nests, Necropolis, and Ice Caves branch from main progression
  * **Safe Levels**: Town levels and some special buildings are marked as safe (no monsters)
  * **Music Themes**: Each area has distinct background music
  * **Size Variations**: Levels have different dimensions, from 16x16 (Spider levels) to 64x64 (Spider Boss)

==== Level Connections ====

The dungeon graph allows for non-linear exploration in some areas:
  * Spider Nest levels form a loop: 6s→7s→8s→9s→10s→6s
  * Necropolis levels form a loop: necro1→necro2→necro3→necro4→necro5→necro1
  * Ice Caves form a loop: ice1→ice2→ice3→ice4→ice5→ice1
  * Each level connects to the next in the main progression
  * Town connects to multiple buildings and Level 1 (Sewer)
