====== Dungeon Layout ======

The Remixed Dungeon has a complex layout with multiple levels and interconnected areas. The dungeon is divided into several themed sections, each with its own challenges and enemies.

==== Main Dungeon Levels ====

The main dungeon progression typically follows this order:

  * [[rpd:town|Town]] - Starting area with shops and services
  * **Sewers** (Levels 1-5) - Early game area with basic enemies
  * **Boss Level** (Level 5) - Features the Goo boss
  * **Prison** (Levels 6-10) - Mid-game area, features Tengu boss
  * **Caves** (Levels 11-15) - Underground caves, features DM-300 boss
  * **City** (Levels 16-20) - Dwarven metropolis, features Dwarven King boss
  * **Halls** (Levels 21-25) - Demon halls, features Yog-Dzewa's guardians
  * **Boss Level** (Level 25) - Features Shadow Lord boss
  * **Guts** (Levels 26-31) - Additional challenging areas
  * **Boss Level** (Level 31) - Features the final Yog-Dzewa boss

==== Special Areas ====

The dungeon also contains special side areas:

  * [[rpd:spider_nest|Spider Nests]] (Levels 6s-10s) - Spider-themed areas
  * [[rpd:necropolis|Necropolis]] - Undead-themed area  
  * [[rpd:ice_caves|Ice Caves]] - Cold-themed area
  * [[rpd:sub_levels|Sub Levels]] - Special themed areas

==== Town Features ====

The [[rpd:town|Town]] area contains several special buildings:

  * **Church** - Healing and blessings
  * **Cinema** - Special area
  * **Inn** - Rest and recovery
  * **Library** - Knowledge and information
  * **Fortune Teller** - Special services
  * **Shop** - Equipment and supplies

==== Level Transitions ====

Each level is connected to others as defined in the dungeon's topology. Players can move up and down between connected levels to explore all areas of the dungeon.
