====== AI Wiki Tools and Suggestions ======

This page documents suggestions for additional tools that could improve the wiki maintenance process for Remixed Dungeon.

==== Suggested Tools ====

  * **Automated Link Checker:** A tool that regularly scans all pages for broken links and missing pages
  * **Image Verification Tool:** Automatically verifies that all image references in wiki pages exist and are accessible
  * **Content Consistency Checker:** Ensures common elements (like stats, descriptions) are consistent across pages and match the game code
  * **Translation Consistency Tool:** Helps maintain consistency across different language versions of the wiki
  * **Auto-Update Tool:** Automatically updates pages that contain frequently changing information like game stats
  * **Content Generation Assistant:** AI-powered tool to help create initial drafts of pages based on game code
  * **Diff Analysis Tool:** Compares changes in game code to existing wiki pages to suggest necessary updates
  * **MR Page Generator:** Tool that automatically generates or updates machine-readable namespace pages (mr:) with code references, JSON configs, and string resources for all game entities
  * **Wiki Verification Assistant:** AI tool that can analyze game source code (Java, Lua, JSON) against wiki content to identify discrepancies or outdated information
  * **Bulk Update Tool:** Tool that can update multiple wiki pages simultaneously when game mechanics change
  * **Naming Convention Enforcer:** Tool to ensure all pages follow the proper naming scheme (_mob, _item, _spell, etc.) and are in lowercase with underscores
  * **Source Code Link Generator:** Tool that automatically adds or updates references to game source code, string resources, and JSON configurations for each wiki entity

==== Current Tools ====

The following tools already exist and are useful:

  * [[https://github.com/NYRDS/remixed-dungeon/blob/master/tools/py-tools/find_red_links.py|find_red_links.py]] - Finds broken links in wiki
  * [[https://github.com/NYRDS/remixed-dungeon/blob/master/tools/py-tools/scale_sprites_for_wiki.py|scale_sprites_for_wiki.py]] - Processes game sprites for wiki use
  * [[https://github.com/NYRDS/remixed-dungeon/blob/master/tools/py-tools/check_broken_images.py|check_broken_images.py]] - Validates image references

==== Best Practices ====

  * Always verify game mechanics against the actual source code
  * Include direct links to relevant code files when documenting features
  * Use consistent naming schemes for game entities
  * Reference string resource names for accuracy of in-game text
  * Regularly run validation tools to maintain quality
  * Cross-reference information from multiple sources (Java, Lua, JSON, string resources) to ensure completeness
  * Create proper machine-readable (mr:) reference pages for all game entities
