====== Necromancer Armor ======

{{ rpd:images:necromancerarmor_sprite.png|Necromancer Armor }}

The **Necromancer Armor** is a class-specific armor that can only be worn by the [[rpd:necromancer_class|Necromancer]] class. When worn, it allows the Necromancer to use the [[rpd:necrotism|Necrotism]] ability, which spreads a magic plague that may infect any nearby creatures (living or dead).

==== Description ====
This armor is specifically designed for the [[rpd:necromancer_class|Necromancer]] class and provides a powerful necromantic ability when worn. It allows the use of the [[rpd:necrotism|Necrotism]] ability, which spreads a magic plague that may infect any nearby creatures (living or dead).

==== Properties ====
  * **Class Restriction:** Only [[rpd:necromancer_class|Necromancer]] can wear this armor
  * **Special Ability:** Allows use of [[rpd:necrotism|Necrotism]] ability
  * **Name:** Also known as "Bone armor"
  * **Armor Type:** Medium armor with moderate defensive capabilities

==== Special Abilities ====
  * **[[rpd:necrotism|Necrotism]]:** When worn, allows the hero to use this special ability that spreads a magic plague to creatures nearby, both living and dead

==== Usage ====
  * To use the [[rpd:necrotism|Necrotism]] ability, equip the armor and select the "Necrotism" action from the menu
  * The ability affects all creatures in close proximity to the Necromancer
  * Uses 3 charges of mana when activated

==== Acquisition ====
  * [[rpd:necromancer_armor_item|Necromancer Armor]] is class-specific and can only be equipped by the [[rpd:necromancer_class|Necromancer]]
  * Obtained by using an [[rpd:armor_kit_item|Armor Kit]] on any regular armor (Body Armor, Mail Armor, Scale Armor, etc.)
  * The [[rpd:armor_kit_item|Armor Kit]] is found on the City level, dropped by the King boss

==== Strategy ====
  * Essential armor for the [[rpd:necromancer_class|Necromancer]] class to access key necromantic abilities
  * [[rpd:necrotism|Necrotism]] is especially effective against hordes of weaker enemies
  * The buff spreads from creature to creature, with each infected creature spreading it further
  * Higher-level creatures are more resistant (take less damage, harder to spread to)
  * Combines defensive capability with powerful area effect ability
