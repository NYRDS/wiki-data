====== KunaiThrow ======

{{ rpd:images:kunai_throw_spell_icon.png|KunaiThrow Spell Icon }}

**KunaiThrow** is a spell in Remixed Pixel Dungeon (Rogue Affinity).

==== Description ====
This Deadly Kunai throw will be done extremely fast, so fast that you won't even be able to select a target.\n\nTrue Masters able to throw more than one Kunai at once.

==== Stats ====
  * **Magic Affinity:** Rogue
  * **Targeting:** self (targets random visible enemies)
  * **Level:** 2
  * **Mana Cost:** 5
  * **Cast Time:** 0.01 (extremely fast)
  * **Special Effects:** damages random visible enemies
  * **Damage:** Random damage between n and (n + caster level * n), where n is the number of kunai thrown
  * **Number of Kunai:** Based on caster skill level (min(caster skill level - 1, 3), max 3 kunai)
  * **Targeting Logic:** The spell automatically targets random enemies visible to the caster
