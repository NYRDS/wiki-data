====== Shaman ======

{{ rpd:images:shaman_mob.png|Shaman }}

==== Description ====
The **Shaman** is a hero subclass for the [[rpd:elf_class|Elf]] class in Remixed Dungeon. The Shaman specializes in using staves as both melee and ranged weapons, with unique mechanics that allow for efficient combat with staff weapons.

==== Stats and Abilities ====
* **Primary Attribute**: Magical combat with staves
* **Special Mechanic**: Can use staves as melee weapons effectively
* **Wand Synergy**: When wielding a staff, wand zaps can be used without losing charges
* **Staff Mastery**: Improved effectiveness when using staff weapons

==== Class Requirements ====
* Requires: [[rpd:elf_class|Elf]] class and reaching the appropriate level
* Alternative name: [[rpd:hero_subclasses#shaman|Shaman Subclass]]

==== Gameplay Strategy ====
* The Shaman subclass works best with staff-based weapons since they can use them effectively in melee combat
* The charge conservation on wands makes this subclass particularly effective for players who rely on wand usage
* Synergizes well with items and mechanics that enhance staff or wand usage
