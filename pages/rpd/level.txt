====== Уровни ======

**Уровни** - это отдельные слои подземелья, на которых происходит игровой процесс.

==== Структура подземелья ====
  * Подземелье состоит из 26 уровней (с первого по 26-й)
  * Каждый уровень представляет собой отдельное подземелье с уникальной генерацией
  * Уровни разделены на несколько [[rpd:areas|зон]]:
    * [[rpd:sewers_level|Канализация]] (уровни 1-5)
    * [[rpd:prison_level|Тюрьма]] (уровни 6-10)
    * [[rpd:caves_level|Пещеры]] (уровни 11-15)
    * [[rpd:city_level|Город]] (уровни 16-20)
    * [[rpd:halls_level|Чертоги]] (уровни 21-25)
    * [[rpd:throne_level|Тронный зал]] (уровень 26)

==== Типы уровней ====
  * [[rpd:random_level|Случайные уровни]] - Стандартные уровни с рандомной генерацией
  * [[rpd:predesigned_level|Предопределённые уровни]] - Уровни с фиксированной структурой
  * [[rpd:boss_level|Уровни боссов]] - Особые уровни с сильным врагом
  * [[rpd:safe_level|Безопасные уровни]] - Уровни без врагов и ловушек

==== Особенности уровней ====
  * [[rpd:stairs|Лестницы]] - Позволяют перемещаться между уровнями
  * [[rpd:shops|Магазины]] - Могут появляться на некоторых уровнях
  * [[rpd:traps|Ловушки]] - Скрытые опасности на большинстве уровней
  * [[rpd:chests|Сундуки]] - Содержат [[rpd:loot|добычу]]
  * [[rpd:terrain|Ландшафт]] - Различные типы местности на уровнях

==== Механики уровней ====
  * [[rpd:lighting|Освещение]] - Различные уровни освещённости
  * [[rpd:view_distance|Дальность обзора]] - Может изменяться на разных уровнях
  * [[rpd:level_generation|Генерация]] - Алгоритмы создания уровней
  * [[rpd:level_mechanics|Механики]] - Специфические особенности каждого уровня

==== Проверка содержимого ====

  * Entity Kind: Each level type implements getEntityKind() method
  * String Resources: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/res/values/strings_all.xml|strings_all.xml]] contains level names and descriptions
  * Configuration: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/levels/|levels/]] directory contains various level implementations
