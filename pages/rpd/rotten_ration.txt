====== Rotten Ration ======

{{ rpd:images:item_RottenRation.png|Rotten Ration }}

Rotten Ration is a spoiled food item in Remixed Dungeon that provides no nutritional benefit.

==== Stats ====
  * **Rarity**: Rare (only created by poisoning regular Rations)
  * **Weight**: 1
  * **Energy**: 0 food units (no hunger restoration)
  * **Consumption Time**: 3 turns
  * **Price**: Cannot be purchased or sold

==== Description ====
Rotten Ration is a spoiled version of the standard Ration. It can only be created when a regular Ration becomes poisoned in harmful environmental conditions. Unlike regular rations, it provides no nutritional benefit when consumed.

==== Usage ====
  * Does not satisfy hunger when consumed
  * Does not prevent starvation damage
  * Takes 3 turns to consume (same as other food items)
  * Cannot be eaten for any benefit

==== Special Properties ====
  * Created only by poisoning regular [[rpd:ration|Rations]]
  * Provides no nutritional benefit when consumed
  * Cannot be found naturally in treasure drops (0 probability in Treasury system)
  * Can be purified back to a normal [[rpd:ration|Ration]]

==== Acquisition ====
  * Created by environmental poisoning of regular Rations
  * Cannot be found naturally in dungeon
  * Cannot be purchased in shops

==== Strategy ====
  * Avoid consuming as it provides no benefit
  * If you have Rotten Rations, consider dropping them or using them for other purposes
  * Can be purified back into normal Rations using the purification method
