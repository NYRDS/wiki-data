====== Arrow ======

{{ rpd:images:arrow_item.png|Arrow }}

The **Arrow** is a basic ranged ammunition used with bows. It forms the base class for specialized arrows.

==== Stats ====
  * **Damage**: 0 (base damage is 0)
  * **Delay**: 1.0 (baseDly: 1.0)
  * **Min Strength Required**: 9 (setSTR: 9)
  * **Default quantity**: Typically ranges from 15-30 when randomly generated

==== Special Abilities ====
  * **Base Class**: Arrow serves as the base class for specialized arrow types like Fire Arrow, Poison Arrow, and Amok Arrow
  * **Mechanism**: Each specialized arrow type adds its own special effect on successful hit
  * **Behavior**: Functions as standard ranged ammunition

==== Acquisition ====
  * Can be found in various treasure locations throughout the dungeon
  * Quantity typically ranges from 15-30 when randomly generated
  * Price: 5 gold per arrow

==== Strategy ====
  * Essential for using with bow weapons
  * Specialized arrows provide different tactical advantages
  * Ammunition management is important for bow users
