====== Arcane Stylus ======

{{ rpd:images:arcane_stylus_item.png|Arcane Stylus }}

The Arcane Stylus is a magical tool used to inscribe special properties onto scrolls in Remixed Dungeon.

==== Stats ====
  * **Rarity**: Rare
  * **Weight**: 1
  * **Type**: Crafting Tool

==== Description ====
The Arcane Stylus allows the user to inscribe additional magical properties onto blank scrolls, transforming them into special scrolls with enhanced powers.

==== Usage ====
  * Combine with blank scrolls to create special scroll variants
  * Used for advanced scroll crafting
  * Requires blank scrolls to function
  * Single-use tool per inscription

==== Acquisition ====
  * Found in magical or scholarly areas of the dungeon
  * May be available in magical shops
  * Relatively rare but valuable item

==== Special Notes ====
  * Essential for creating special scroll variants
  * Requires blank scrolls to use
  * Can create more powerful scroll effects
  * Valuable for spell-focused characters
  * Part of advanced magical crafting mechanics
