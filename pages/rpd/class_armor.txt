====== Class Armor and Special Abilities ======

Each playable class in Remixed Pixel Dungeon has a distinctive class armor with unique special abilities that can be used in combat. Class armor is obtained by using an [[rpd:armor_kit|Armor Kit]] on any regular armor, which transforms it into the class-specific armor variant.

==== Warrior Armor: Heroic Leap ====

The Warrior's class armor enables a special ability called **Heroic Leap**:

  * **Ability**: Leap to a targeted location and create a shockwave that stuns all adjacent enemies
  * **Effect**: Allows the warrior to reposition in combat and disable multiple enemies briefly
  * **Strategy**: Useful for escaping dangerous situations or setting up advantageous positioning
  * **Source Code Reference**: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/items/armor/WarriorArmor.java|WarriorArmor.java]]

==== Mage Armor: Molten Earth ====

The Mage's class armor ability provides enhanced spellcasting capabilities via **Molten Earth**:

  * **Ability**: Sets all enemies in field of view on fire and roots them in place
  * **Effect**: Inflicts burning damage over time and disables movement of affected enemies
  * **Strategy**: Excellent for crowd control and area denial
  * **Source Code Reference**: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/items/armor/MageArmor.java|MageArmor.java]]

==== Rogue Armor: Smoke Bomb ====

The Rogue's class armor focuses on mobility and evasion via **Smoke Bomb**:

  * **Ability**: Blinds enemies who can see the rogue and teleports to a targeted location
  * **Effect**: Provides temporary invisibility and repositioning capabilities
  * **Strategy**: Supports the Rogue's hit-and-run combat style and escape mechanisms
  * **Source Code Reference**: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/items/armor/RogueArmor.java|RogueArmor.java]]

==== Huntress Armor: Spectral Blades ====

The Huntress's armor channels natural forces through **Spectral Blades**:

  * **Ability**: Creates a fan of spectral blades that target all visible enemies
  * **Effect**: Each blade inflicts damage depending on the huntress's currently equipped melee weapon
  * **Strategy**: Effective for dealing area damage to multiple enemies simultaneously
  * **Source Code Reference**: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/items/armor/HuntressArmor.java|HuntressArmor.java]]

==== Elf Armor: Nature Call ====

The Elf's armor combines magic and physical defense with **Nature Call**:

  * **Ability**: Summons high grass growth throughout the area, including through solid stone
  * **Effect**: Immobilizes all enemies in sight and provides tactical advantages through terrain manipulation
  * **Strategy**: Excellent for controlling enemy movement and creating advantageous terrain
  * **Source Code Reference**: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/items/armor/ElfArmor.java|ElfArmor.java]]

==== Necromancer Armor: Necrotism ====

The Necromancer's armor has unique necromantic properties through **Necrotism**:

  * **Ability**: Activate a necrotic field that spreads to enemies
  * **Effect**: Creates a spreading death effect that can damage multiple enemies and affect dead creatures
  * **Strategy**: Supports the Necromancer's undead summoning and necromantic abilities
  * For more details, see [[rpd:necromancer_armor|Necromancer Armor]]
  * **Source Code Reference**: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/nyrds/pixeldungeon/items/common/armor/NecromancerArmor.java|NecromancerArmor.java]]

==== Gnoll Armor: Dominate ====

The Gnoll's armor reflects their pack-based nature with **Dominate**:

  * **Ability**: Attempts to dominate enemy creatures to become pets
  * **Effect**: Converts enemy mobs to fight alongside the gnoll up to a limit based on hero level
  * **Strategy**: Supports the Gnoll's pet-based combat approach and army building
  * **Source Code Reference**: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/items/armor/GnollArmor.java|GnollArmor.java]]

==== Priest Armor ====

The Priest's armor focuses on purification and has no active ability:

  * **Ability**: No special active ability
  * **Effect**: Provides standard defensive properties for the Priest class
  * **Strategy**: Focuses on passive support through other class mechanics
  * **Source Code Reference**: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/nyrds/pixeldungeon/items/common/armor/PriestArmor.java|PriestArmor.java]]

==== Doctor Coat ====

The Doctor's armor reflects their expertise in harmful substances but has no active ability:

  * **Ability**: No special active ability
  * **Effect**: Provides standard defensive properties for the Doctor class
  * **Strategy**: Focuses on passive combat through other class mechanics
  * **Source Code Reference**: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/nyrds/pixeldungeon/items/common/armor/DoctorArmor.java|DoctorArmor.java]]

==== Strategy Tips ====

  * Class armor abilities typically require skill points (mana) to use
  * Each armor ability is unique to its class and complements the class's playstyle
  * Class armor is obtained by using an [[rpd:armor_kit|Armor Kit]] on any regular armor
  * Higher-level class abilities can significantly impact combat outcomes
  * All class armors are only usable by their respective classes
