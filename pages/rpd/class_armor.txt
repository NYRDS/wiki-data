====== Class Armor ======

==== Description ====
**Class Armor** refers to the unique, powerful armor sets available to each hero class in Remixed Dungeon. These armors provide special abilities and bonuses in addition to standard defensive properties. Each class has a unique class armor that unlocks at tier 2 of that class's armor tree.

==== Class-Specific Armors ====
* [[rpd:warrior_armor_item|Warrior Armor]] → [[rpd:gladiator_armor|Gladiator Armor]] (Warrior subclass)
* [[rpd:mage_armor_item|Mage Armor]] → [[rpd:battle_mage_armor|Battle Mage Armor]] (Mage subclass)
* [[rpd:rogue_armor_item|Rogue Armor]] → [[rpd:assassin_armor|Assassin Armor]] (Rogue subclass)
* [[rpd:huntress_armor_item|Huntress Armor]] → [[rpd:freerunner_armor|Freerunner Armor]] (Huntress subclass)
* [[rpd:elf_armor|Elf Armor]] → [[rpd:shaman_armor|Shaman Armor]] (Elf subclass)
* [[rpd:necromancer_armor|Necromancer Armor]] → [[rpd:witchdoctor_armor|Witchdoctor Armor]] (Necromancer subclass)
* [[rpd:doctor_armor|Doctor Armor]] → [[rpd:plague_doctor|Plague Doctor Armor]] (Doctor subclass)
* [[rpd:priest_armor|Priest Armor]] → [[rpd:warden_armor|Warden Armor]] (Priest subclass)
* [[rpd:gnoll_armor|Gnoll Armor]] → [[rpd:berserker_armor|Berserker Armor]] (Gnoll subclass)

==== Special Abilities ====
Each class armor provides unique special abilities:
  * [[rpd:gladiator_armor|Gladiator]]: Throw weapon and return it to the hero
  * [[rpd:assassin_armor|Assassin]]: Blink to a targeted location
  * [[rpd:freerunner_armor|Freerunner]]: Phase through walls and gain speed boost
  * [[rpd:battle_mage_armor|Battle Mage]]: Empower the staff with magical energy
  * [[rpd:shaman_armor|Shaman]]: Special staff-based mechanics
  * [[rpd:witchdoctor_armor|Witchdoctor]]: Necromancy-related abilities
  * [[rpd:plague_doctor|Plague Doctor]]: Disease and healing mechanics
  * [[rpd:warden_armor|Warden]]: Terrain manipulation abilities
  * [[rpd:berserker_armor|Berserker]]: Rage and health conversion abilities

==== Obtaining ====
Class armor can be obtained in two ways:
  1. **Crafting**: Using an [[rpd:armor_kit|Armor Kit]] on regular armor of the same tier
  2. **Random drops**: Found as rare drops from various sources throughout the dungeon
