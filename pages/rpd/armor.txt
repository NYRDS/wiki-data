====== Armor ======

{{ rpd:images:armor_Leather.png|Armor }}

**Armor** is an equipment type in Remixed Dungeon that provides defense against physical attacks.

==== Base Types ====
  * [[rpd:cloth_armor_item|Cloth Armor]] - The basic armor available to all heroes
  * [[rpd:leather_armor_item|Leather Armor]] - Crafted armor made from tanned monster leather
  * [[rpd:mail_armor_item|Mail Armor]] - Interlocking metal rings for improved protection
  * [[rpd:scale_armor_item|Scale Armor]] - Metal scales attached to a leather backing
  * [[rpd:plate_armor_item|Plate Armor]] - Heavy metal plates for maximum protection

==== Class Armor ====
Each hero class has access to a unique class armor that can be crafted using an [[rpd:armor_kit_item|Armor Kit]]:
  * [[rpd:warrior_armor_item|Warrior Armor]]
  * [[rpd:mage_armor_item|Mage Armor]]
  * [[rpd:rogue_armor_item|Rogue Armor]]
  * [[rpd:huntress_armor_item|Huntress Armor]]
  * [[rpd:elf_armor_item|Elf Armor]]
  * [[rpd:necromancer_robe_item|Necromancer Robe]]
  * [[rpd:gnoll_armor_item|Gnoll Armor]]

==== Special Armor ====
  * [[rpd:spider_armor_item|Spider Armor]] - Crafted using spider silk
  * [[rpd:gothic_armor_item|Gothic Armor]] - High-tier armor with special properties

==== Mechanics ====
  * **Defense**: Armor provides a defense value that reduces incoming physical damage
  * **Armor Penetration**: Some weapons and attacks can partially ignore armor
  * **Enchantments**: Armor can be enchanted with [[rpd:glyphs|glyphs]] that provide special effects
  * **Durability**: Armor has a charge system that can be maintained with dew

==== Crafting ====
  * Using an [[rpd:armor_kit_item|Armor Kit]] on any regular armor creates the class-specific armor for that hero
  * The [[rpd:the_king|King]] drops armor kits that allow the creation of class armors
  * Higher-tier armor requires more resources and rare materials
