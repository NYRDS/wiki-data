====== Мобы ======

**Мобы** (мобильные объекты) - это враждебные и нейтральные существа, с которыми герой сталкивается в подземелье.

==== Основные типы мобов ====
  * [[rpd:rats|Крысы]] - Простые враги на начальных уровнях
  * [[rpd:gnoll_mob|Гноллы]] - Существа с различными специализациями
  * [[rpd:crab_mob|Крабы]] - Сильные враги с высокой защитой
  * [[rpd:spider_mob|Пауки]] - Ядовитые существа, распространённые в пещерах
  * [[rpd:elemental_mob|Элементали]] - Магические существа различных стихий
  * [[rpd:undead_mobs|Нежить]] - Существа, уязвимые к святой воде
  * [[rpd:mechanical_enemies|Механические враги]] - Роботы и механизмы
  * [[rpd:bosses|Боссы]] - Мощные враги, требующие особых стратегий

==== Статистика и механики ====
  * **Здоровье:** Текущее и максимальное количество здоровья
  * **Урон:** Урон, который моб может нанести герою
  * **Защита:** Способность избежать урона
  * **Скорость:** Влияет на частоту действий моба
  * **ИИ:** Поведение моба в бою и при перемещении

==== Специальные атаки ====
  * [[rpd:ranged_enemies|Дальнобойные враги]] - Мобы, способные атаковать на расстоянии
  * [[rpd:mechanical_enemies|Механические враги]] - Имеют особые свойства
  * [[rpd:aoe_attacks|Атаки по области]] - Поражают несколько целей

==== Поведение ====
  * **Агрессия:** Как мобы реагируют на присутствие героя
  * **Патрулирование:** Движение моба по территории
  * **Отступление:** Условия, при которых моб убегает

==== Проверка содержимого ====
  * Java Class: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/actors/mobs/Mob.java|Mob.java]] (базовый класс)
  * Entity Kind: Each mob has its own getEntityKind() implementation
  * String Resources: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/res/values/strings_all.xml|strings_all.xml]] contains mob names and descriptions
  * Configuration: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/assets/mobsDesc/|mobsDesc/]] directory contains JSON configuration for different mobs
