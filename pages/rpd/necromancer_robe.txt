====== Necromancer Robe ======

{{ rpd:images:necromancerarmor_sprite.png|Necromancer Robe Sprite }}

The Necromancer Robe is specialized equipment for the [[rpd:necromancer|Necromancer]] class.

==== Description ====
This robe is imbued with dark magic and will allow the [[rpd:necromancer|Necromancer]] to accumulate skill points from defeated monsters. When any creature dies while the Necromancer is wearing this robe, the wearer gains 1 skill point.

==== Properties ====
  * **Class Restriction:** Only [[rpd:necromancer|Necromancers]] can use this robe
  * **Armor Class:** Functions as a regular armor piece (tier 1)
  * **Soul Accumulation:** Provides 1 skill point when any creature dies while wearing the robe

==== Abilities ====
  * **Soul Accumulation:** Gain 1 skill point when any creature dies while wearing the robe

==== Acquisition ====
  * Found as treasure in dungeon levels
  * Potentially available in special shops

==== Strategy ====
  * Provides passive skill point accumulation for Necromancer heroes
  * Good for long-term progression and character development
  * Allows for skill point farming by luring enemies to their death
  * Works well in combination with other necromantic abilities for sustained combat
