====== Skeleton Key ======

{{ rpd:images:skeleton_key_item.png|Skeleton Key }}

The **Skeleton Key** is a unique key item that opens the path to the Demon Halls.

==== Description ====
This ancient key was crafted from a demon's bone, and allows access to the depths of the dungeon where powerful demons dwell. It is an essential item for accessing the final area of the dungeon.

==== Properties ====
  * **Type:** Key Item
  * **Rarity:** Unique
  * **Slot:** None (consumed when used)
  * **Class:** All

==== Usage ====
  * **Primary Function:** Opens the entrance to the [[rpd:halls|Demon Halls]]
  * **Consumed:** Yes, when used to unlock the Demon Halls
  * **Special:** Only works once to unlock the Demon Halls area

==== Acquisition ====
  * **Source:** Dropped by [[rpd:tengu_mob|Tengu]] in the Prison area
  * **Drop Rate:** 100% (guaranteed drop from Tengu)
  * **Availability:** Only obtainable once per game

==== Mechanics ====
  * When used in the appropriate location, permanently unlocks the Demon Halls area
  * After unlocking, the key is consumed and removed from inventory
  * Critical item for progression to the end-game content

==== Notes ====
  * Essential for reaching end-game content
  * Should be preserved until reaching the appropriate location
  * Once used, cannot be replaced
