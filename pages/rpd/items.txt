====== Предметы ======

**Предметы** - это объекты, которые герой может подбирать, использовать и носить с собой в [[rpd:inventory|инвентаре]].

==== Категории предметов ====
  * [[rpd:weapons|Оружие]] - Позволяет наносить урон врагам
  * [[rpd:armor|Броня]] - Уменьшает получаемый урон
  * [[rpd:accessories|Аксессуары]] - [[rpd:rings|Кольца]] и другие вспомогательные предметы
  * [[rpd:potions|Зелья]] - Эффекты при употреблении
  * [[rpd:scrolls|Свитки]] - Магические эффекты при прочтении
  * [[rpd:wands|Палочки]] - Перезаряжаемые магические предметы
  * [[rpd:food|Еда]] - Восполняет голод
  * [[rpd:keys|Ключи]] - Открывают замки
  * [[rpd:spells|Заклинания]] - Магические способности
  * [[rpd:artifacts|Артефакты]] - Уникальные экипируемые предметы

==== Общие свойства предметов ====
  * [[rpd:identification|Опознание]] - Необходимо для выяснения свойств предмета
  * [[rpd:upgrades|Улучшения]] - Повышение характеристик предмета
  * [[rpd:enchantments|Зачарования]] - Специальные свойства оружия и брони
  * [[rpd:weight|Вес]] - Влияет на переносимую нагрузку
  * [[rpd:price|Цена]] - Стоимость в магазине

==== Редкость предметов ====
  * **Обычные** - Встречаются часто
  * **Необычные** - Более редкие предметы
  * **Редкие** - Особые мощные предметы
  * **Эпические** - Очень редкие и сильные предметы
  * **Легендарные** - Уникальные предметы с особыми эффектами

==== Состояние предметов ====
  * [[rpd:curse|Проклятие]] - Негативные эффекты при экипировке
  * [[rpd:durability|Прочность]] - Некоторые предметы могут ломаться
  * [[rpd:charges|Заряды]] - Для магических предметов

==== Проверка содержимого ====
  * Java Class: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/java/com/watabou/pixeldungeon/items/Item.java|Item.java]] (базовый класс)
  * Entity Kind: Various item types implement getEntityKind() method
  * String Resources: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/res/values/strings_all.xml|strings_all.xml]] contains item name and description strings
  * Configuration: [[https://github.com/NYRDS/remixed-dungeon/blob/master/RemixedDungeon/src/main/assets/itemsDesc/|itemsDesc/]] directory contains JSON configuration for different items
